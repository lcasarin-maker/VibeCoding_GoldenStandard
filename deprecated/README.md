# deprecated/

This folder contains historical artifacts from the Golden Standard's development.
These files are **not active knowledge** — they are preserved for traceability and context.

## Structure

| Folder | Contents |
|---|---|
| `mandates_legacy/` | Early mandate drafts (S01–S23) before they were integrated into the YAML catalogs |
| `knowledge_snapshots/` | Audit reports, library snapshots, and correlation matrices from development phases |
| `wiki_phases/` | Phase completion documents (PHASE_2A, 2C, 3) tracking the Wiki's build-out |
| `planning/` | Original planning, audit guides, normalization mandates from the GS construction process |

## Policy

- Files here are **read-only reference**. Do not modify them.
- If you find a concept here that should be promoted to active knowledge, open an Issue and link to the relevant file.
- Nothing in `deprecated/` is enforced by any audit or validation tool.
