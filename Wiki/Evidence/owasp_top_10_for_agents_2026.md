# OWASP Top 10 for Agents 2026

> Citable evidence source referenced by 10 entries in the Golden Standard catalog.

## Claims

- insecure default configuration is a frequent vector in AI-generated code.
- insecure handling of tool outputs is a recognized agentic risk.
- persistent memory poisoning is a recognized agentic risk.
- prompt injection / tool misuse is the #1 risk of agentic applications.
- recognized risk category (input validation / broken authorization) for agentic applications.
