### Symptom (Signal)
> Expensive context is used to decide

### Cause
Incorrect routing

### Application / Mitigation
Separate thinking, executing, and reviewing

### Operational Relevance
Separate thinking, executing, and reviewing so expensive execution context is not spent deciding; the canonical guidance must stay published and linked from the index, which a CI link-check verifies.


### ❌ Example of the vice (Bad)
```text
# The expensive execution context is used to deliberate decisions.
```

### ✅ Corrected version (Good)
```text
# Separate thinking / executing / reviewing into distinct spaces.
```

### 🔎 Concrete detection
Deliberating using the expensive execution context instead of separating phases.

### 📚 External evidence
- **[[Evidence/vexp_dev_humanlayer_ace|vexp.dev / HumanLayer ACE]]** — unstructured context consumes 3-5x more tokens; selective retrieval, caching, and compaction are the central levers.
