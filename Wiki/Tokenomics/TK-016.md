### Symptom (Signal)
> The model splits its attention

### Cause
Objective overload

### Application / Mitigation
One concrete task per cycle

### Operational Relevance
Pursue one concrete task per cycle so attention is not split across objectives; the canonical guidance must stay published and linked from the index, which a CI link-check verifies.


### ❌ Example of the vice (Bad)
```text
# One prompt asks for refactor + tests + docs + deploy at once.
```

### ✅ Corrected version (Good)
```text
# One concrete task per cycle; the model does not split attention.
```

### 🔎 Concrete detection
Multi-objective prompts that scatter attention instead of one task per cycle.

### 📚 External evidence
- **vexp.dev / HumanLayer ACE** — unstructured context consumes 3-5x more tokens; selective retrieval, caching, and compaction are the central levers.
