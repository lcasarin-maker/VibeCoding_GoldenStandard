### Symptom (Signal)
> Command output saturates the context

### Cause
Operational noise

### Application / Mitigation
Compress or filter before handing off to the agent

### Operational Relevance
Long-running background loops and orchestrators must route output through a compression/throttling layer; static analysis flags scripts that do not wrap execution in an output compressor.


### ❌ Example of the vice (Bad)
```bash
pytest -v                       # dumps thousands of raw lines into the context
```

### ✅ Corrected version (Good)
```bash
pytest -q | tail -20            # filter/compress before handing off to the agent
```

### 🔎 Concrete detection
Raw command output dumped into the context without filtering or compression.

### 📚 External evidence
- **[[Evidence/vexp_dev_humanlayer_ace|vexp.dev / HumanLayer ACE]]** — unstructured context consumes 3-5x more tokens; selective retrieval, caching, and compaction are the central levers.
