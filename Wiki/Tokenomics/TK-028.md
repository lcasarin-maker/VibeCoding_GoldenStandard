### Symptom (Signal)
> The user counts messages

### Cause
Fragile human control

### Application / Mitigation
Visible and automatable thresholds

### Operational Relevance
Make thresholds visible and automatable so the user does not manually count messages; the canonical measurement-and-telemetry guidance must stay published and linked from the index, which a CI link-check verifies.


### ❌ Example of the vice (Bad)
```text
# The user counts messages by hand to know when to compact.
```

### ✅ Corrected version (Good)
```text
# Visible and automatable thresholds for context usage.
```

### 🔎 Concrete detection
Manual/forgettable context monitoring instead of visible thresholds.

### 📚 External evidence
- **[[Evidence/vexp_dev_humanlayer_ace|vexp.dev / HumanLayer ACE]]** — unstructured context consumes 3-5x more tokens; selective retrieval, caching, and compaction are the central levers.
