### Síntoma (Signal)
> Formato variable

### Causa (Cause)
Distribución de salida amplia

### Aplicación / Mitigación
Predefinir inicio y estructura

### Relevancia Operativa
Output-compaction topology validation keeps the compaction lens and its index published.

### ❌ Ejemplo del vicio (Bad)
```python
resp = call(model, prompt)              # formato de salida variable
```

### ✅ Versión corregida (Good)
```python
resp = call(model, prompt, prefill="{\"result\":")  # estructura predefinida
```

### 🔎 Detección concreta
Salida sin prefilling/estructura predefinida cuando el formato debe ser estable.

### 📚 Evidencia externa
- **vexp.dev / HumanLayer ACE** — el contexto sin estructura consume 3-5x mas tokens; recuperacion selectiva, caching y compactacion son las palancas centrales.
