| **Severity** | **low** |
| **Depth** | 🟢 Deep |
| **Tags** | `tokenomics`, `prevented` |
| **Downstream Verification** | `none` |
| **Validation Mechanism** | `runtime-test` |

---

### Symptom (Signal)
> Variable format

### Cause
Wide output distribution

### Application / Mitigation
Predefine the start and structure

### Operational Relevance
Predefine the start and structure of responses so the format is not variable; the canonical output-and-compaction guidance must stay published and linked from the index, which a CI link-check verifies.


### ❌ Example of the vice (Bad)
```python
resp = call(model, prompt)              # variable output format
```

### ✅ Corrected version (Good)
```python
resp = call(model, prompt, prefill="{\"result\":")  # predefined structure
```

### 🔎 Concrete detection
Output without prefilling/predefined structure when the format must be stable.

### 📚 External evidence
- **vexp.dev / HumanLayer ACE** — unstructured context consumes 3-5x more tokens; selective retrieval, caching, and compaction are the central levers.
