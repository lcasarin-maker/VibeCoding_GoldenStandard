### Symptom (Signal)
> Continuity requires reading the full history

### Cause
Non-normalized state

### Application / Mitigation
Atomic handoff: objective, state, evidence, risks, next action

### Operational Relevance
Session handoff state must use structured sections rather than free prose so it is machine-parseable; a test fails on unstructured handoff.


### ❌ Example of the vice (Bad)
```text
# Long-prose handoff: continuing requires reading the entire history.
```

### ✅ Corrected version (Good)
```text
# Atomic handoff: objective | state | evidence | risks | next action.
```

### 🔎 Concrete detection
Handoffs in extensive prose without an atomic structure of key fields.

### 📚 External evidence
- **vexp.dev / HumanLayer ACE** — unstructured context consumes 3-5x more tokens; selective retrieval, caching, and compaction are the central levers.
