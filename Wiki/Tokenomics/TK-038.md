### Symptom (Signal)
> Each turn drags everything along

### Cause
Non-indexed state

### Application / Mitigation
Keep small state fields

### Operational Relevance
D10 manifest size gate validates that AGENT.md <= 150 lines, STATUS.md <= 200 lines, and SPEC.md <= 500 lines.

### ❌ Example of the vice (Bad)
```python
ctx = read_full_state()         # each turn drags the full state along
```

### ✅ Corrected version (Good)
```python
ctx = read_state_fields(['goal','blockers'])   # small fields
```

### 🔎 Concrete detection
Re-reading the full state every turn instead of small fields.

### 📚 External evidence
- **vexp.dev / HumanLayer ACE** — unstructured context consumes 3-5x more tokens; selective retrieval, caching, and compaction are the central levers.
