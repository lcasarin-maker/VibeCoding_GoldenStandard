### Síntoma (Signal)
> Cada turno arrastra todo

### Causa (Cause)
Estado no indexado

### Aplicación / Mitigación
Mantener campos pequeños de estado

### Relevancia Operativa
D10 manifest size gate validates that AGENT.md <= 150 lines, STATUS.md <= 200 lines, and SPEC.md <= 500 lines.

### ❌ Ejemplo del vicio (Bad)
```python
ctx = read_full_state()         # cada turno arrastra el estado completo
```

### ✅ Versión corregida (Good)
```python
ctx = read_state_fields(['goal','blockers'])   # campos pequenos
```

### 🔎 Detección concreta
Releer el estado completo cada turno en vez de campos pequenos.

### 📚 Evidencia externa
- **vexp.dev / HumanLayer ACE** — el contexto sin estructura consume 3-5x mas tokens; recuperacion selectiva, caching y compactacion son las palancas centrales.
