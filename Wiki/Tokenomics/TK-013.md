### Symptom (Signal)
> Long responses by default

### Cause
Unbounded output space

### Application / Mitigation
Define a budget and format

### Operational Relevance
Define an output budget and format so responses are not long by default; the canonical output-and-compaction guidance must stay published and linked from the index, which a CI link-check verifies.


### 🔎 Concrete detection
Output with no defined budget or format, long by default.
