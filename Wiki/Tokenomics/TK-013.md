### Síntoma (Signal)
> Herramientas no usadas consumen contexto

### Causa (Cause)
Carga anticipada excesiva

### Aplicación / Mitigación
Carga de herramientas diferida

### Relevancia Operativa
Automation-tooling topology validation keeps the tooling surface explicit so unused tools do not become invisible context debt.

### ❌ Ejemplo del vicio (Bad)
```python
tools = load_all_tools()        # esquemas de 40 tools no usados ocupan contexto
```

### ✅ Versión corregida (Good)
```python
tools = load_tools_for(task)    # carga diferida segun necesidad
```

### 🔎 Detección concreta
Cargar esquemas de herramientas no usadas en el contexto en vez de carga diferida.

### 📚 Evidencia externa
- **vexp.dev / HumanLayer ACE** — el contexto sin estructura consume 3-5x mas tokens; recuperacion selectiva, caching y compactacion son las palancas centrales.
