### Symptom (Signal)
> Redundant reprocessing of stable specifications

### Cause
Confusion between logical invariants and session variables

### Application / Mitigation
Separate guidelines into an Immutable Core and compact the Transitional Buffer

### Operational Relevance
Separate stable guidelines into an immutable core and compact the transitional buffer so stable specs are not reprocessed; the canonical memory-and-headroom guidance must stay published and linked from the index, which a CI link-check verifies.


### ❌ Example of the vice (Bad)
```text

```

### ✅ Corrected version (Good)

```text
# Separate Immutable Core (stable) and compact the Transitional Buffer (volatile).
```

### 🔎 Concrete detection
Reprocessing stable guidelines alongside volatile context without segregating memory.
