### Síntoma (Signal)

### Causa (Cause)
Autorización no estructurada

### Aplicación / Mitigación
Representar permisos como matriz consultable

### Relevancia Operativa
Behavioral/doctrinal tokenomics vice — not statically falsifiable in a generic way. Documented in the Golden Standard catalogs as governance knowledge; no automated test can discriminate this without human semantic judgment. Sprint 3.4 triage: reclassified from AUDITED/test_d10_tokenomics to DOC_ONLY.

### ❌ Ejemplo del vicio (Bad)
```text
# Politica de permisos narrada en prosa, repetida en cada prompt.
"El rol admin puede X pero no Y; el rol lector solo Z; el rol auditor..."
```

### ✅ Versión corregida (Good)
```text
# Matriz consultable; el prompt referencia, no narra.
permisos[rol][accion] -> allow | deny
```

### 🔎 Detección concreta
Politicas de autorizacion narradas en prosa dentro de prompts en vez de una
matriz/tabla consultable.
