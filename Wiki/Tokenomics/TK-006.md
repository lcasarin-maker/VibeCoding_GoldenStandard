### Symptom (Signal)
> Resolving a conflict consumes excessive context

### Cause
Non-semantic memory

### Application / Mitigation
Merge by decisions and invariants, not by raw text

### Operational Relevance
validate_golden_standard_catalogs.py --check-wiki keeps the canonical tokenomics map and memory lenses linked so history is not merged blindly.

### ❌ Example of the vice (Bad)
```text
# Resolve the history conflict by re-reading all the raw text.
```

### ✅ Corrected version (Good)
```text
# Merge by decisions and invariants, not line-by-line text.
```

### 🔎 Concrete detection
Merging history/state by raw text instead of by decisions and invariants.

### 📚 External evidence
- **vexp.dev / HumanLayer ACE** — unstructured context consumes 3-5x more tokens; selective retrieval, caching, and compaction are the central levers.
