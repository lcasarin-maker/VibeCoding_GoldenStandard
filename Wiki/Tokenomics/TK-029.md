### Symptom (Signal)
> Each turn drags everything along

### Cause
Non-indexed state

### Application / Mitigation
Keep small state fields

### Operational Relevance
Core memory manifests must stay under fixed line budgets so context is not saturated; a static size gate fails when a manifest exceeds its budget.


### ❌ Example of the vice (Bad)
```python
ctx = read_full_state()         # each turn drags the full state along
```

### ✅ Corrected version (Good)
```python
ctx = read_state_fields(['goal','blockers'])   # small fields
```

### 🔎 Concrete detection
Re-reading the full state every turn instead of small fields.

### 📚 External evidence
- **[[Evidence/vexp_dev_humanlayer_ace|vexp.dev / HumanLayer ACE]]** — unstructured context consumes 3-5x more tokens; selective retrieval, caching, and compaction are the central levers.
