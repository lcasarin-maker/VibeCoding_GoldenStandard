# CD12 — Dependency & Supply-Chain Posture

Keeps declared dependencies, package freshness, and third-party intake aligned with the real code surface and vulnerability posture.

---

## Graph Role

Supply-chain traceability hub: connects doctrine to package reality and prevents invisible third-party drift.

---

## Covers

- Declared versus imported dependency reconciliation
- SCA and yanked-package posture
- Third-party code and package intake discipline

---

## Explicitly Excludes

- Generic runtime exfiltration logic
- Assertion quality
- Session telemetry
