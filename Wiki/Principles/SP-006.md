## Doctrine

| Field | Value |
|---|---|
| **Symptom** | CC, GS, and satellite repos diverge in folder structure, file naming, or governance layout |
| **Cause** | No automated enforcement; each repo evolves independently |
| **Solution** | scripts/audit.py enforces the canonical tree; a failing audit blocks session close in any repo |
| **Action** | audit.py is the single structural enforcer; its checks define what the canonical structure IS. |

---

## Detection

audit.py: validate required folders and files exist; run across CC, GS, and registered satellites
