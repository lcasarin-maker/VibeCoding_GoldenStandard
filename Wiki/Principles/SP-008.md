## Doctrine

| Field | Value |
|---|---|
| **Symptom** | Tasks stall in active/ waiting for human intervention; or agents declare done without meeting DoD |
| **Cause** | Unclear authority over task completion; no explicit checklist |
| **Solution** | Agent moves task to done/ when AGENTS.md §4 DoD checklist is fully met; review/ is used only for tasks explicitly flagged as requiring human acceptance |
| **Action** | AGENTS.md §4 DoD must be a concrete checklist; any item not met = task stays active or moves to blocked/. |
