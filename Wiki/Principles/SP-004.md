## Doctrine

| Field | Value |
|---|---|
| **Symptom** | A done/ or archive/ folder accumulates completed task files as a parallel changelog |
| **Cause** | Fear of losing record; agents keep files "just in case" |
| **Solution** | Completed task file is deleted after its closing commit; the commit message is the durable record; no archive/ folder |
| **Action** | Delete task file in the same commit that closes the work; write a descriptive commit message. |

---

## Detection

audit.py: assert no archive/ folder exists; assert tasks/done/ is empty
