## Doctrine

| Field | Value |
|---|---|
| **Symptom** | Task files and audit session records use ad-hoc formats; fields missing; records not comparable across sessions |
| **Cause** | No template enforcement; each agent invents a format |
| **Solution** | tasks/README.md and audit/README.md define mandatory templates; audit.py checks compliance |
| **Action** | Create tasks/README.md and audit/README.md with required field templates; add compliance check to audit.py. |

---

## Detection

audit.py: assert tasks/README.md and audit/README.md exist; assert session files contain required sections
