# PI-001: Insight Satélite

### Descripción del Insight
> Deptry – reconciliación de imports contra dependencias declaradas para detectar faltantes, no usadas, transitivas, dev mal ubicadas y stdlib declaradas como dependencias.

---

### Mapeo a Dominios de Auditoría Cerberus
*   **[[Domains/D1|D1]]** (Proyecto: *deptry*): Compare imports against declared dependencies and fail on missing, unused, transitive or misplaced packages.
*   **[[Domains/D2|D2]]** (Proyecto: *deptry*): Treat missing or stale dependency declarations as completeness debt and block delivery until reconciled.
*   **[[Domains/D8|D8]]** (Proyecto: *deptry*): Prevent dependency drift from destabilizing the test suite by validating imports before running coverage gates.
