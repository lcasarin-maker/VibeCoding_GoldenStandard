### Symptom

### Cause
Exempt debt

### Solution
Validate critical flows

### Corrective Action / Prevention
Validate critical UI flows directly, not only visual deltas that miss functional breakage.


### ❌ Example of the vice (Bad)
```python

```

### ✅ Corrected version (Good)
```python
# validate critical flows: click -> state -> result
```

### 🔎 Concrete detection
