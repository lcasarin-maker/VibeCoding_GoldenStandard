### Symptom
> Operation depends on a fixed location

### Cause
Hardcoded external state

### Solution
Resolve locations by configurable contract

### Corrective Action / Prevention
Filesystem paths must be portable: a scan fails when a hardcoded absolute path (drive letter or home directory) appears in scripts.


### ❌ Example of the vice (Bad)
```python
# Absolute path from the agent's machine: breaks in any other environment.
LOG = open("C:/Users/Luis/AI/project/out.log", "w")
```

### ✅ Corrected version (Good)
```python
# Path derived from configuration/environment and relative to the project root.
from pathlib import Path
import os
LOG = open(Path(os.environ.get("PROJECT_ROOT", ".")) / "out.log", "w")
```

### 🔎 Concrete detection
Regex/AST: path literals containing 'C:\\', '/Users/', '/home/' or
Windows drive separators in application code.
