### Symptom
> The test compares against an expectation that no longer matches the current specification

### Cause
The specification or the reference fixture became stale

### Solution
Review the source of truth for the expectation and update the oracle

### Corrective Action / Prevention
Audited as stale-oracle mismatch; separate from copied-expected contamination.

### ❌ Example of the vice (Bad)
```python
assert tax(100) == 7   # the spec changed to 8%, the test is still at 7
```

### ✅ Corrected version (Good)
```python
assert tax(100) == SPEC.rate * 100
```

### 🔎 Concrete detection
Oracle that no longer matches the current specification.
