### Symptom
> The file exists

### Cause
Weak proxy

### Solution
Validate the effect

### Corrective Action / Prevention
Asserting that a file or artifact merely exists is not a test; validate its effect or content.


### ❌ Example of the vice (Bad)
```python
assert os.path.exists("out.csv")   # exists, but the content is not validated
```

### ✅ Corrected version (Good)
```python
assert read_csv("out.csv").rows == 3   # observable effect
```

### 🔎 Concrete detection
Tests that assert the existence of an artifact without validating its effect/content.
