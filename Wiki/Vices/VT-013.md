### Symptom
> Gamified coverage

### Cause
Wrong incentive

### Solution
Assertion quality

### Corrective Action / Prevention
Coverage percentage is not a goal; what matters is the discriminating quality of the assertions, not a gamed number.


### ❌ Example of the vice (Bad)
```python
# Empty tests created only to reach the % coverage threshold.
def test_a(): import mod          # 'covers' the import
def test_b(): mod.helper()        # calls without verifying
```

### ✅ Corrected version (Good)
```python
# The goal is verified behavior, not the coverage number.
def test_helper_doubles():
    assert mod.helper(2) == 4
```

### 🔎 Concrete detection
Detect tests whose declared purpose/commit is 'raise coverage' and whose body
lacks asserts on results.
