### Symptom
> The MCP server declares wildcard permissions or executes unregistered capabilities.

### Cause
Evasion of least-privilege policies.

### Solution
Statically verify that every executed capability is listed in the manifest.

### Corrective Action / Prevention
Enforced by mapping code capabilities to manifest in dimensions/d15_agent_security.py.

### ❌ Example of the vice (Bad)
```json
{ "permissions": ["*"] }
```

### ✅ Corrected version (Good)
```json
{ "permissions": ["read_file:docs/"] }
```

### 🔎 Concrete detection
Compares call usage against the manifest's permission schema.

### 📚 External evidence
- **NVIDIA SkillSpector mcp_least_privilege.py** — Wildcard permissions nullify the security granularity of tool servers.
