### Síntoma (Symptom)
> El servidor de MCP declara permisos comodín o ejecuta capacidades no registradas.

### Causa (Cause)
Evasión de políticas de menor privilegio.

### Solución (Solution)
Verificar estáticamente que toda capacidad ejecutada esté listada en el manifiesto.

### Acción Correctiva / Prevención
Enforced by mapping code capabilities to manifest in dimensions/d15_agent_security.py.

### ❌ Ejemplo del vicio (Bad)
```json
{ "permissions": ["*"] }
```

### ✅ Versión corregida (Good)
```json
{ "permissions": ["read_file:docs/"] }
```

### 🔎 Detección concreta
Compara el uso de llamadas con el esquema de permisos del manifiesto.

### 📚 Evidencia externa
- **NVIDIA SkillSpector mcp_least_privilege.py** — Los permisos comodín anulan la granularidad de seguridad de los servidores de herramientas.
