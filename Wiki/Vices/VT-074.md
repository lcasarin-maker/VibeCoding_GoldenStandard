### Symptom
> On failure there is no useful diagnostic

### Cause
Logs outside the quality criterion

### Solution
Evaluate that the failure produces actionable evidence

### Corrective Action / Prevention
Test that a failure produces actionable diagnostic evidence, not silence.


### ❌ Example of the vice (Bad)
```python
# nobody verifies that a failure produces an actionable log
```

### ✅ Corrected version (Good)
```python
def test_failure_logs(): with fail(): assert log_has("cause")
```

### 🔎 Concrete detection
Missing test to verify that failures produce actionable evidence.
