### Symptom
> Only the permitted actor is tested

### Cause
Denial not exercised

### Solution
Validate positive and negative permissions

### Corrective Action / Prevention
Validate both positive and negative permission cases, not only the permitted actor.


### ❌ Example of the vice (Bad)
```python
assert can_read(admin)   # only positive cases
```

### ✅ Corrected version (Good)
```python
assert can_read(admin) and not can_read(guest)   # positive and negative
```

### 🔎 Concrete detection
Permission tests that cover only allowed accesses, not denied ones.
