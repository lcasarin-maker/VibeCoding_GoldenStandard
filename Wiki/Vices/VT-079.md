### Symptom
> Isolation is declared without attempting an escape

### Cause
Threat not induced

### Solution
Test controlled boundary violations

### Corrective Action / Prevention
Test isolation by attempting controlled boundary violations, not by merely declaring the sandbox secure.


### ❌ Example of the vice (Bad)
```python
# it is never tested that the sandbox blocks forbidden accesses
```

### ✅ Corrected version (Good)
```python
# test controlled boundary violations (they must fail)
```

### 🔎 Concrete detection
Sandbox/security without tests that exercise boundary violations.
