### Symptom
> The test compares against an answer written to pass

### Cause
Oracle derived from the implementation

### Solution
Derive expected values from an independent specification

### Corrective Action / Prevention
Derive expected values from an independent specification, not an answer written so the test passes.


### ❌ Example of the vice (Bad)
```python
EXPECTED = run(inp)   # the expected value comes from the SUT
```

### ✅ Corrected version (Good)
```python
# derive the expected value from an independent specification
```

### 🔎 Concrete detection
Expected value derived from the SUT instead of an independent specification.
