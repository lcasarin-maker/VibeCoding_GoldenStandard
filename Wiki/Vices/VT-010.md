### Symptom
> Internals change, it still passes

### Cause
Incorrect coupling

### Solution
Test behavior

### Corrective Action / Prevention
Audited by DeepForensicAuditor D8 and D9 behavioral and static test validations.

### ❌ Example of the vice (Bad)
```python
assert obj._helper.call_count == 1   # couples to internal detail
```

### ✅ Corrected version (Good)
```python
assert obj.result() == 42   # observable behavior
```

### 🔎 Concrete detection
Tests that assert on internal details (private calls) instead of behavior.
