### Symptom
> Internals change, it still passes

### Cause
Incorrect coupling

### Solution
Test behavior

### Corrective Action / Prevention
Test observable behavior, not internal implementation detail, so a behavior-preserving refactor keeps the test meaningful.


### ❌ Example of the vice (Bad)
```python
assert obj._helper.call_count == 1   # couples to internal detail
```

### ✅ Corrected version (Good)
```python
assert obj.result() == 42   # observable behavior
```

### 🔎 Concrete detection
Tests that assert on internal details (private calls) instead of behavior.
