### Síntoma (Symptom)
> Marcador satisface aserción

### Causa (Cause)
Ausencia confundida con implementación

### Solución (Solution)
Bloquear placeholders en rutas verificadas

### Acción Correctiva / Prevención
Prevented by D7 (Completeness) and D8 (Test Coverage) in run_security_audit_12d.py using AST analysis of function bodies to reject empty stubs or stub docstrings.

### ❌ Ejemplo del vicio (Bad)
```python
# Se testea el valor placeholder en vez del comportamiento real.
def get_price(): return 0.0   # placeholder
def test_price(): assert get_price() == 0.0
```

### ✅ Versión corregida (Good)
```python
def get_price(): return catalog.lookup(sku).price
def test_price(): assert get_price() == 19.99
```

### 🔎 Detección concreta
Tests que afirman exactamente valores placeholder conocidos (0, '', 'TODO', None)
devueltos por implementaciones no terminadas.

### 📚 Evidencia externa
- **arxiv:2602.00409** — estudio empirico confirma que los agentes generan tests sin oraculo (sin assert, tautologicos o sobre-mockeados) de forma medible.
