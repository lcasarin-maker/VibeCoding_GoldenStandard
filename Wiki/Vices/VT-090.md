### Symptom
> A placeholder satisfies the assertion

### Cause
Absence mistaken for implementation

### Solution
Block placeholders on verified paths

### Corrective Action / Prevention
Block placeholders on verified paths; a placeholder that satisfies the assertion fakes completion.


### ❌ Example of the vice (Bad)
```python
# The placeholder value is tested instead of the real behavior.
def get_price(): return 0.0   # placeholder
def test_price(): assert get_price() == 0.0
```

### ✅ Corrected version (Good)
```python
def get_price(): return catalog.lookup(sku).price
def test_price(): assert get_price() == 19.99
```

### 🔎 Concrete detection
Tests that assert exactly known placeholder values (0, '', 'TODO', None)
returned by unfinished implementations.
