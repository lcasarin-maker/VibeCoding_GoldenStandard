### Symptom
> A process stays blocked undetected

### Cause
Lack of a liveness signal

### Solution
Require a heartbeat, timeout, and recovery policy

### Corrective Action / Prevention
Behavioral/doctrinal vice — not statically falsifiable in a generic way. Documented in the Golden Standard catalogs as governance knowledge; no automated test can discriminate this without human semantic judgment. Sprint 3.4 triage: reclassified from AUDITED/test_behavioral_compliance to DOC_ONLY.

### ❌ Example of the vice (Bad)
```python
resp = sock.recv(1024)        # no timeout: if the peer dies, it hangs forever
```

### ✅ Corrected version (Good)
```python
sock.settimeout(30)
resp = sock.recv(1024)        # timeout + retry/recovery policy
```

### 🔎 Concrete detection
Blocking calls (recv/lock.acquire/join/await) without a configured timeout or
heartbeat.
