### Symptom
> Verifies a universal law

### Cause
Useless predicate

### Solution
The test must discriminate

### Corrective Action / Prevention
A test must discriminate the subject's behavior, not restate a universal truth that holds regardless of the code.


### ❌ Example of the vice (Bad)
```python
def test_norm():
    x = normalize(payload)
    assert x == x   # tautology: true for any value
```

### ✅ Corrected version (Good)
```python
def test_norm():
    assert normalize({"A": 1}) == {"a": 1}   # oracle independent of the SUT
```

### 🔎 Concrete detection
AST: asserts whose two operands are the same expression (assertEqual(x, x),
assert x == x).
