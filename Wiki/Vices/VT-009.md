### Síntoma (Symptom)
> Verifica ley universal

### Causa (Cause)
Predicado inútil

### Solución (Solution)
Test debe discriminar

### Acción Correctiva / Prevención
Checked by D9 (Test Purity) using AST TestTheaterVisitor to flag assert True, assertTrue(True), assertEqual(x, x), and tests without active asserts.

### ❌ Ejemplo del vicio (Bad)
```python
def test_norm():
    x = normalize(payload)
    assert x == x   # tautologia: cierto para cualquier valor
```

### ✅ Versión corregida (Good)
```python
def test_norm():
    assert normalize({"A": 1}) == {"a": 1}   # oraculo independiente del SUT
```

### 🔎 Detección concreta
AST: asserts cuyos dos operandos son la misma expresion (assertEqual(x, x),
assert x == x).

### 📚 Evidencia externa
- **arxiv:2602.00409** — estudio empirico confirma que los agentes generan tests sin oraculo (sin assert, tautologicos o sobre-mockeados) de forma medible.
