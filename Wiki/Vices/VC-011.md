### Symptom
> Code is replaced without diagnosis

### Cause
Erasure of causality

### Solution
Diagnosis before generation

### Corrective Action / Prevention
Regeneration must precede validation: a fresh rebuild of the vault is forced before checks, so blind replacement cannot masquerade as a finished state.


### ❌ Example of the vice (Bad)
```python
# The whole function is replaced without diagnosing why it failed.
def parse(x): return brand_new_impl(x)   # 'regenerate until it passes'
```

### ✅ Corrected version (Good)
```python
# Diagnose first: reproduce, isolate cause, fix minimally.
def parse(x): return x.strip() or DEFAULT   # fix aimed at the cause
```

### 🔎 Concrete detection
Total replacement of a failing unit without evidence of prior diagnosis (repro/cause).
