### Symptom
> Passes due to prior state

### Cause
Broken isolation

### Solution
Independent tests

### Corrective Action / Prevention
Tests must be independent; one that passes only because of another's leftover state is unreliable.


### ❌ Example of the vice (Bad)
```python
# test_b depends on the state left by test_a
```

### ✅ Corrected version (Good)
```python
# each test creates and cleans up its own state (independent)
```

### 🔎 Concrete detection
Tests that depend on the execution order (state shared between tests).
