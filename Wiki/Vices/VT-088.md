### Symptom
> The system accepts a known error

### Cause
Infinite-exception policy

### Solution
Zero accepted errors in a health verdict

### Corrective Action / Prevention
A health verdict accepts zero known errors; tolerating one normalizes it.


### ❌ Example of the vice (Bad)
```python
# The test tolerates errors: any result passes.
def test_sync():
    result = sync()
    assert result is not None or result is None   # always true
```

### ✅ Corrected version (Good)
```python
def test_sync():
    assert sync().synced_count == 10
```

### 🔎 Concrete detection
AST: asserts whose condition is a logical tautology (X or not X) or that accept
any value from the SUT.
