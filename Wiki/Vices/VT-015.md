### Symptom
> Does not localize the failure

### Cause
Low granularity

### Solution
Focal tests

### Corrective Action / Prevention
A test should localize the failure it detects; an over-broad test that cannot point to the cause has little diagnostic value.


### ❌ Example of the vice (Bad)
```python
def test_everything(): assert run_whole_app() == "ok"   # does not localize the failure
```

### ✅ Corrected version (Good)
```python
# focal tests per unit: test_parse, test_validate, test_render
```

### 🔎 Concrete detection
A monolithic test that covers the whole pipeline and does not localize the failure to a unit.
