### Symptom
> Does not localize the failure

### Cause
Low granularity

### Solution
Focal tests

### Corrective Action / Prevention
Audited by DeepForensicAuditor D8 and D9 behavioral and static test validations.

### ❌ Example of the vice (Bad)
```python
def test_everything(): assert run_whole_app() == "ok"   # does not localize the failure
```

### ✅ Corrected version (Good)
```python
# focal tests per unit: test_parse, test_validate, test_render
```

### 🔎 Concrete detection
A monolithic test that covers the whole pipeline and does not localize the failure to a unit.

- **arxiv:2602.00409** — an empirical study confirms that agents generate tests without an oracle (no assert, tautological, over-mocked) in a measurable way.
