### Symptom
> Confirms their own model

### Cause
Author bias

### Solution
Independent review

### Corrective Action / Prevention
Tests benefit from independent review; an author validating only their own mental model confirms bias.


### ❌ Example of the vice (Bad)
```text
# The author writes the code and their only reviewer is themselves.
```

### ✅ Corrected version (Good)
```text
# independent review (another actor or model)
```

### 🔎 Concrete detection
Implementation reviewed/tested only by its author, without an independent oracle.
