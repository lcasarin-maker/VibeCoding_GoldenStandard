### Symptom
> The test disables checks

### Cause
Privileged environment

### Solution
Equivalent config

### Corrective Action / Prevention
Run with a configuration equivalent to production; an environment variable that disables checks weakens the test.


### ❌ Example of the vice (Bad)
```python
if os.getenv("TEST"): skip_validation()   # validation differs in test
```

### ✅ Corrected version (Good)
```python
# equivalent configuration: same validation path
```

### 🔎 Concrete detection
Environment variables that alter the validation path in tests.
