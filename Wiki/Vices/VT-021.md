### Síntoma (Symptom)
> Error viejo reaparece

### Causa (Cause)
Memoria de fallos no convertida en prueba

### Solución (Solution)
Cada defecto corregido deja prueba discriminante

### Acción Correctiva / Prevención
Audited by DeepForensicAuditor D8 and D9 behavioral and static test validations.

### ❌ Ejemplo del vicio (Bad)
```python
# se arregla BUG-9 pero no se deja test que lo discrimine
```

### ✅ Versión corregida (Good)
```python
def test_bug9(): assert safe_divide(1, 0) is None   # centinela de regresion
```

### 🔎 Detección concreta
Defecto corregido sin un test discriminante que impida su reaparicion.

### 📚 Evidencia externa
- **arxiv:2602.00409** — estudio empirico confirma que los agentes generan tests sin oraculo (sin assert, tautologicos, sobre-mockeados) de forma medible.
