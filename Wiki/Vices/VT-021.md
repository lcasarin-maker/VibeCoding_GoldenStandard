### Symptom
> An old error reappears

### Cause
Failure memory not turned into a test

### Solution
Every fixed defect leaves a discriminating test

### Corrective Action / Prevention
Every fixed defect must leave behind a discriminating regression test so the same error cannot silently reappear.


### ❌ Example of the vice (Bad)
```python
# BUG-9 is fixed but no test is left to discriminate it
```

### ✅ Corrected version (Good)
```python
def test_bug9(): assert safe_divide(1, 0) is None   # regression sentinel
```

### 🔎 Concrete detection
Defect fixed without a discriminating test that prevents its reappearance.

### 📚 External evidence
- **arxiv:2602.00409** — an empirical study confirms that agents generate tests without an oracle (no assert, tautological, over-mocked) in a measurable way.
