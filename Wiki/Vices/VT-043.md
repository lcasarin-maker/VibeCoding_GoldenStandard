### Symptom
> Green pipeline

### Cause
False exit code

### Solution
Exit reflects the real state

### Corrective Action / Prevention
The exit code must reflect the real state; an unconditional success exit produces a falsely green pipeline.


### ❌ Example of the vice (Bad)
```python
run_checks(); sys.exit(0)   # exit 0 no matter what
```

### ✅ Corrected version (Good)
```python
sys.exit(0 if all_ok else 1)
```

### 🔎 Concrete detection
Unconditional exit code 0 that does not reflect the real state.
