### Síntoma (Symptom)
> Ejecuta sin verificar

### Causa (Cause)
Métrica vacía

### Solución (Solution)
Cobertura con oráculos

### Acción Correctiva / Prevención
Checked by D9 (Test Purity) using AST TestTheaterVisitor to flag assert True, assertTrue(True), assertEqual(x, x), and tests without active asserts.

### ❌ Ejemplo del vicio (Bad)
```python
# Llama a todo para subir coverage, pero no afirma nada.
def test_pipeline():
    run_stage_a(); run_stage_b(); run_stage_c()   # 100% lineas, 0% verificacion
```

### ✅ Versión corregida (Good)
```python
def test_pipeline():
    out = run_stage_c(run_stage_b(run_stage_a(seed)))
    assert out.status == "done" and out.rows == 3
```

### 🔎 Detección concreta
Cruzar cobertura con densidad de asserts: archivos con alta cobertura y tests sin
asserts activos = cobertura inflada.

### 📚 Evidencia externa
- **arxiv:2602.00409** — estudio empirico confirma que los agentes generan tests sin oraculo (sin assert, tautologicos o sobre-mockeados) de forma medible.
