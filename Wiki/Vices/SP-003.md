### Symptom
> tasks/backlog/ accumulates historical, completed, or stale items alongside live work

### Cause
No discipline on removal; items added but never cleared

### Solution
backlog/ contains only work not yet started; completed items are deleted (record is the git commit)

### Corrective Action / Prevention
On task completion, delete task file from done/ after commit; git log is the archive.

### 🔎 Concrete detection
audit.py: assert tasks/done/ is empty (files moved there are deleted after commit)
