### Symptom
> Tests are gamed

### Cause
Inverted incentive

### Solution
One hundred percent only as a consequence

### Corrective Action / Prevention
Audited by DeepForensicAuditor D8 and D9 behavioral and static test validations.

### ❌ Example of the vice (Bad)
```python
# empty tests added only to reach 100% coverage
```

### ✅ Corrected version (Good)
```python
# 100% is a consequence of covering behavior, not the goal
```

### 🔎 Concrete detection
Creating tests to reach 100% coverage as a goal in itself.
