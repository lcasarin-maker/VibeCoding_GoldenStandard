### Symptom
> Tests are gamed

### Cause
Inverted incentive

### Solution
One hundred percent only as a consequence

### Corrective Action / Prevention
One hundred percent coverage is a consequence of good tests, never the goal; treating it as the target invites gaming.


### ❌ Example of the vice (Bad)
```python
# empty tests added only to reach 100% coverage
```

### ✅ Corrected version (Good)
```python
# 100% is a consequence of covering behavior, not the goal
```

### 🔎 Concrete detection
Creating tests to reach 100% coverage as a goal in itself.
