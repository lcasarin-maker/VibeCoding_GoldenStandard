### Symptom
> The test branch is never entered

### Cause
Invalid control flow

### Solution
Verify the test's execution

### Corrective Action / Prevention
Verify that the test's branch actually executes; a condition that is never entered tests nothing.


### ❌ Example of the vice (Bad)
```python
if 1 == 2:
    assert check()   # dead branch: never runs
```

### ✅ Corrected version (Good)
```python
assert check()
```

### 🔎 Concrete detection
Assertions after an impossible condition (dead branch) that never executes.
