### Síntoma (Symptom)
> Rama de test nunca entra

### Causa (Cause)
Control inválido

### Solución (Solution)
Verificar ejecución del test

### Acción Correctiva / Prevención
Audited by DeepForensicAuditor D8 and D9 behavioral and static test validations.

### ❌ Ejemplo del vicio (Bad)
```python
if 1 == 2:
    assert check()   # rama muerta: nunca corre
```

### ✅ Versión corregida (Good)
```python
assert check()
```

### 🔎 Detección concreta
Aserciones tras una condicion imposible (rama muerta) que nunca se ejecuta.
