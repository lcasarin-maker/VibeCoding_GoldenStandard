### Symptom
> Changes approved blindly

### Cause
Incomplete gate

### Solution
Review the tests

### Corrective Action / Prevention
Review the tests as part of the change; approving code without reviewing its tests is blind.


### ❌ Example of the vice (Bad)
```text
# The PR is approved without checking whether the tests prove anything.
```

### ✅ Corrected version (Good)
```text
# the review includes the quality of the tests
```

### 🔎 Concrete detection
Reviews that approve code without evaluating its tests.

### 📚 External evidence
- **[[Evidence/arxiv2602_00409|arxiv:2602.00409]]** — an empirical study confirms that agents generate tests without an oracle (no assert, tautological, over-mocked) in a measurable way.
