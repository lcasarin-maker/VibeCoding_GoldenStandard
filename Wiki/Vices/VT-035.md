### Symptom
> Ignored failure

### Cause
Normalized debt

### Solution
Prohibit xfail as evidence of health

### Corrective Action / Prevention
An expected-failure marker is not evidence of health; a permanent xfail hides an ignored failure.


### ❌ Example of the vice (Bad)
```python
@pytest.mark.xfail   # marked as an expected failure... forever
def test_refund():
    assert refund(order) == 0
```

### ✅ Corrected version (Good)
```python
# If the bug was fixed, the xfail must drop; if it persists, it carries a date and ticket.
@pytest.mark.xfail(reason="BUG-123, review 2026-07", strict=True)
def test_refund():
    assert refund(order) == 0
```

### 🔎 Concrete detection
Detect @xfail/@expectedFailure without reason+date/ticket or without strict=True
(silent xpass).
