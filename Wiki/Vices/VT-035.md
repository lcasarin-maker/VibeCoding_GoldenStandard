### Síntoma (Symptom)
> Falla ignorada

### Causa (Cause)
Deuda normalizada

### Solución (Solution)
Prohibir xfail como evidencia de salud

### Acción Correctiva / Prevención
Checked by D9 Test Purity which rejects permanent xfail or skip markers unless annotated with removal criteria or reasons.

### ❌ Ejemplo del vicio (Bad)
```python
@pytest.mark.xfail   # marcado como fallo esperado... para siempre
def test_refund():
    assert refund(order) == 0
```

### ✅ Versión corregida (Good)
```python
# Si el bug se arreglo, el xfail debe caer; si sigue, lleva fecha y ticket.
@pytest.mark.xfail(reason="BUG-123, revisar 2026-07", strict=True)
def test_refund():
    assert refund(order) == 0
```

### 🔎 Detección concreta
Detectar @xfail/@expectedFailure sin reason+fecha/ticket o sin strict=True
(xpass silencioso).
