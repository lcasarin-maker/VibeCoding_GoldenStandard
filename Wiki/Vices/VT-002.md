### Symptom
> A fake body passes

### Cause
Missing implementation

### Solution
Block active stubs

### Corrective Action / Prevention
Prevented by D7 (Completeness) and D8 (Test Coverage) in the AST completeness gate using analysis of function bodies to reject empty stubs or stub docstrings.

### ❌ Example of the vice (Bad)
```python
# Permanent stub shipped to production as if it were implemented.
def charge(amount):
    pass   # TODO: implement (6 months and counting)
```

### ✅ Corrected version (Good)
```python
# Real implementation; if not ready, it fails loudly, not silently.
def charge(amount):
    raise NotImplementedError("charge() pending - do not use in prod")
```

### 🔎 Concrete detection
AST: functions whose body is only 'pass'/'...'/docstring (stub) that are not
marked as abstract nor raise NotImplementedError.
