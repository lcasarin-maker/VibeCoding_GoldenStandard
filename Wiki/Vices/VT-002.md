### Síntoma (Symptom)
> Cuerpo falso pasa

### Causa (Cause)
Implementación ausente

### Solución (Solution)
Bloquear stubs activos

### Acción Correctiva / Prevención
Prevented by D7 (Completeness) and D8 (Test Coverage) in run_security_audit_12d.py using AST analysis of function bodies to reject empty stubs or stub docstrings.

### ❌ Ejemplo del vicio (Bad)
```python
# Stub permanente enviado a produccion como si estuviera implementado.
def charge(amount):
    pass   # TODO: implementar (lleva 6 meses)
```

### ✅ Versión corregida (Good)
```python
# Implementacion real; si no esta lista, falla ruidosamente, no en silencio.
def charge(amount):
    raise NotImplementedError("charge() pendiente - no usar en prod")
```

### 🔎 Detección concreta
AST: funciones con cuerpo unicamente 'pass'/'...'/docstring (stub) que no estan
marcadas como abstractas ni lanzan NotImplementedError.
