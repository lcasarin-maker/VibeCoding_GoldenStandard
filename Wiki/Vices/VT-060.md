### Symptom
> Unrealistic state

### Cause
Hidden preparation

### Solution
Test a dirty state

### Corrective Action / Prevention
Test from a realistic, dirty state; over-cleaning setup creates conditions production never sees.


### ❌ Example of the vice (Bad)
```python
def setup(): db.reset(); db.seed_perfect()   # never tests a dirty state
```

### ✅ Corrected version (Good)
```python
# also test realistic dirty/partial states
```

### 🔎 Concrete detection
Setup that sanitizes the state so that dirty states are never tested.
