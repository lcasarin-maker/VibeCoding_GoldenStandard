### Symptom
> Non-existence is declared without searching

### Cause
Insufficient exploration

### Solution
Exhaust the structural search

### Corrective Action / Prevention
Missing files must fail loud: broken-link validation fails when a referenced markdown or wiki target does not resolve on disk.


### ❌ Example of the vice (Bad)
```bash
echo "config.yaml does not exist"      # after looking only at the cwd
```

### ✅ Corrected version (Good)
```bash
find . -name "config*.y*ml"        # exhaust the structural search before declaring
```

### 🔎 Concrete detection
Declaring a file non-existent without a prior recursive/structural search.
