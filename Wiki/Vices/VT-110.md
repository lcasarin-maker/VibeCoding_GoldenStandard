### Symptom
> Scattering of governance, security, backup, or state metadata across multiple hidden folders at the repository root (e.g. `.backup_old`, `.temp_meta`, `.rules_draft`), cluttering the workspace.

### Cause
Lack of a unified view and accumulation of historical dead naming.

### Solution
Radically consolidate all metadata under a single hidden directory of centralized governance (e.g. `.protocol/`).

### Corrective Action / Prevention
Consolidate governance and state metadata under a single hidden directory rather than scattering it across many.


### ❌ Example of the vice (Bad)
```text
# .audit/ .cerberus/ .protocol/ scattered without consolidation.
```

### ✅ Corrected version (Good)
```text
# consolidate metadata under a single hidden directory
```

### 🔎 Concrete detection
Metadata fragmented across multiple dot-directories without consolidation.
