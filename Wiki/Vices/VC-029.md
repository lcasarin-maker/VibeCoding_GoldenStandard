### Symptom
> Duplication of failures

### Cause
Transfer without comprehension

### Solution
Audit reuse

### Corrective Action / Prevention
Behavioral/doctrinal vice — not statically falsifiable in a generic way. Documented in the Golden Standard catalogs as governance knowledge; no automated test can discriminate this without human semantic judgment. Sprint 3.4 triage: reclassified from AUDITED/test_behavioral_compliance to DOC_ONLY.

### ❌ Example of the vice (Bad)
```python
# Same block copied across several handlers; a bug replicates N times.
def a(): u = db.get(id); log(u); return u.name
def b(): u = db.get(id); log(u); return u.email
```

### ✅ Corrected version (Good)
```python
# Common logic extracted just once.
def _load(id): u = db.get(id); log(u); return u
def a(): return _load(id).name
def b(): return _load(id).email
```

### 🔎 Concrete detection
Clone detection: blocks of >=5 identical lines in >=2 locations
(jscpd / pylint duplicate-code).
