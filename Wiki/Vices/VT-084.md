### Symptom
> The approved state is set manually as truth

### Cause
Non-verifiable approval authority

### Solution
Approval based on reproducible evidence

### Corrective Action / Prevention
Base approval on reproducible evidence, not a manually-set approved state treated as truth.


### ❌ Example of the vice (Bad)
```python
APPROVED = True   # fixed approval
```

### ✅ Corrected version (Good)
```python
APPROVED = evidence_reproducible()
```

### 🔎 Concrete detection
Hardcoded approval instead of based on reproducible evidence.
