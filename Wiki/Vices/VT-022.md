### Symptom
> Useless passing tests that do not discriminate logical breaks

### Cause
Coverage gamification prioritizing a syntactic metric over rigor

### Solution
Require boundary-value assertions and prohibit global catches that silence failures

### Corrective Action / Prevention
Require boundary-value assertions and forbid broad catches that silence failures, so a green run cannot be ceremonial.


### ❌ Example of the vice (Bad)
```python
# Guaranteed 'green': a combination of tautological asserts.
def test_calc():
    assert 1 == 1
    assert isinstance(calc(2), int)   # does not fix the value
```

### ✅ Corrected version (Good)
```python
def test_calc():
    assert calc(2) == 4
```

### 🔎 Concrete detection
AST: tests whose asserts are all tautological or type-based (isinstance) without fixing
any concrete expected value.
