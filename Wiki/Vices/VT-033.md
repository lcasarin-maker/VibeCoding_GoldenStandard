### Symptom
> A layer is created so the validator does not fail

### Cause
Hiding a broken contract

### Solution
Fix the cause in the subject, do not adapt the judge

### Corrective Action / Prevention
Fix the cause in the subject under test; adding a layer so the judge stops failing adapts the judge, not the defect.


### ❌ Example of the vice (Bad)
```python
def safe_audit(x):
    try: return audit(x)
    except: return 'ok'   # adapts the judge
```

### ✅ Corrected version (Good)
```python
# fix the cause in the subject, do not wrap the auditor
```

### 🔎 Concrete detection
Wrapper that adapts the evaluator to hide the failure instead of fixing the subject.
