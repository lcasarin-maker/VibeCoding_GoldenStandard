### Síntoma (Symptom)
> UNREADABLE parece dato

### Causa (Cause)
Error codificado como valor

### Solución (Solution)
Fallo explícito

### Acción Correctiva / Prevención
Audited by DeepForensicAuditor D8 and D9 behavioral and static test validations.

### ❌ Ejemplo del vicio (Bad)
```python
data = read()
assert data   # data == "UNREADABLE" pasa como verdadero
```

### ✅ Versión corregida (Good)
```python
data = read()

```

### 🔎 Detección concreta
