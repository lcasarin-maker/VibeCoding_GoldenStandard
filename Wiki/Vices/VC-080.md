### Symptom
> Tool calls with dangerous parameters that evade validations.

### Cause
Indirect injection through tool parameters.

### Solution
Strictly validate and type the tools' arguments.

### Corrective Action / Prevention
Tool calls must be constrained: static analysis enforces parameter limits and rejects unsafe tool chaining.


### ❌ Example of the vice (Bad)
```python
run_command(user_input, shell=True)
```

### ✅ Corrected version (Good)
```python
run_command(["ls", sanitized_directory_path])
```

### 🔎 Concrete detection
Maps the use of shell=True and direct passing of variables to commands.
