### Symptom
> Tool descriptions contain covert instructions or homoglyphs.

### Cause
Use of metadata channels to inject instructions.

### Solution
Metadata sanitization and Unicode similarity validation.

### Corrective Action / Prevention
Tool descriptions must be trustworthy: static analysis scans for unicode homoglyphs and poisoned tool-description content.


### ❌ Example of the vice (Bad)
```python
def format_tеxt(input_str):  # 'е' is a Unicode homoglyph (Cyrillic Small Letter Ie)
    pass
```

### ✅ Corrected version (Good)
```python
def format_text(input_str):
    pass
```

### 🔎 Concrete detection
Searches for homoglyphs in identifiers and invisible text/directives in metadata.
