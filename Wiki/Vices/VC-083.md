### Symptom
> Undeclared artifacts survive

### Cause
Non-binding inventory

### Solution
Every dead file must be authorized, classified, or deleted

### Corrective Action / Prevention
Detected by D1 audit_d1_integrity: scans all code files (excluding non-code by extension) and fails with "Unregistered file (dead)" if the file is not in the whitelist nor a legitimate test. Upgrade 2026-06-04: VC-083 leaves DOC_ONLY because D1 already verifies it mechanically.

### ❌ Example of the vice (Bad)
```text
# old_server.py, server_v2_backup.py coexist without being declared or used.
```

### ✅ Corrected version (Good)
```text
# Each artifact is authorized, classified, or deleted (explicit inventory).
```

### 🔎 Concrete detection
Files not referenced by imports/build nor declared in an inventory = dead files.
