### Symptom
> outcome always success

### Cause
Predetermined result

### Solution
Outcome derived from verification

### Corrective Action / Prevention
The recorded outcome must be derived from an actual verification, not a hardcoded success.


### ❌ Example of the vice (Bad)
```python
def run(): return {"outcome": "success"}   # constant
```

### ✅ Corrected version (Good)
```python
def run(): return {"outcome": verify().status}
```

### 🔎 Concrete detection
Constant outcome/verdict instead of derived from verification.
