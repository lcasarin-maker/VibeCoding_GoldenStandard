### Symptom
> The protocol fails partially without reporting which dependency is missing

### Cause
Preconditions not verified at startup; the system assumes a complete environment

### Solution
Active stack validation at startup: runtime, CLI tools, hooks, essential files, write permissions

### Corrective Action / Prevention
Validate the full active stack at startup - runtime, CLI tools, hooks, essential files, write permissions - and report exactly what is missing.


### ❌ Example of the vice (Bad)
```python
# It swallows part of the stack: the test passes even though a stage failed silently.
def test_flow():
    try:
        a(); b(); c()
    except StepError:
        pass            # 'c' may not have run and nobody knows
```

### ✅ Corrected version (Good)
```python
def test_flow():
    assert a().ok
    assert b().ok
    assert c().ok       # each stage verified, no silent capture
```

### 🔎 Concrete detection
try/except that spans multiple steps and absorbs intermediate errors without
reasserting that each step completed.
