# VT-106: Exclusión no revalidada

| Campo | Detalle |

---

### Síntoma (Symptom)
> Lista de ignorados no tiene test que verifique que cada exclusión sigue siendo válida

### Causa (Cause)
Deriva silenciosa: lo que se ignoró puede dejar de ser inofensivo; nadie lo detecta

### Solución (Solution)
Test periódico que comprueba que cada ítem de hard_excludes está en el conjunto aprobado y documentado

### Acción Correctiva / Prevención
Checked by setup_validate.py which runs comprehensive pre-flight verification of Python, git hooks, write access, encoding, and the project registry.
