### Symptom
> Always true

### Cause
Tautology

### Solution
Require a demonstrable failure

### Corrective Action / Prevention
An assertion must be able to fail: a check that is always true is rejected as non-discriminating.


### ❌ Example of the vice (Bad)
```python
def test_login():
    user = login("a", "b")
    assert True    # trivial assert: always passes, does not verify the login
```

### ✅ Corrected version (Good)
```python
def test_login():
    user = login("a", "b")
    assert user.is_authenticated and user.name == "a"
```

### 🔎 Concrete detection
AST TestTheaterVisitor: presence of 'assert True' / 'assertTrue(True)' as the test's
only assertion.
