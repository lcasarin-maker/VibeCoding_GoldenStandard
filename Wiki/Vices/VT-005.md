### Síntoma (Symptom)
> Siempre verdadero

### Causa (Cause)
Tautología

### Solución (Solution)
Exigir fallo demostrable

### Acción Correctiva / Prevención
Checked by D9 (Test Purity) using AST TestTheaterVisitor to flag assert True, assertTrue(True), assertEqual(x, x), and tests without active asserts.

### ❌ Ejemplo del vicio (Bad)
```python
def test_login():
    user = login("a", "b")
    assert True    # assert trivial: pasa siempre, no verifica el login
```

### ✅ Versión corregida (Good)
```python
def test_login():
    user = login("a", "b")
    assert user.is_authenticated and user.name == "a"
```

### 🔎 Detección concreta
AST TestTheaterVisitor: presencia de 'assert True' / 'assertTrue(True)' como unica
asercion del test.

### 📚 Evidencia externa
- **arxiv:2602.00409** — estudio empirico confirma que los agentes generan tests sin oraculo (sin assert, tautologicos o sobre-mockeados) de forma medible.
