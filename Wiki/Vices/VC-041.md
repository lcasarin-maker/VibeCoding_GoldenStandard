### Symptom
> The system presumes the environment exists

### Cause
Unverified preconditions

### Solution
Validate execution requirements before operating

### Corrective Action / Prevention
Setup must be real, not phantom: inbox topology validation requires the ingestion protocol, knowledge-sources contract, and templates to exist before new findings are accepted.


### ❌ Example of the vice (Bad)
```python
open('./data/cache.db')          # assumes ./data exists; fails in a clean environment
```

### ✅ Corrected version (Good)
```python
Path('./data').mkdir(exist_ok=True)   # preflight: validate/create requirements before operating
open('./data/cache.db', 'a')
```

### 🔎 Concrete detection
Code that presumes directories/services/variables without validating them at startup.
