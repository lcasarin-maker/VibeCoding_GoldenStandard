### Symptom
> The loop improves nothing

### Cause
False semantic contract

### Solution
Name matching behavior

### Corrective Action / Prevention
A name must match behavior; a function whose name promises work its body never does is misleading.


### ❌ Example of the vice (Bad)
```python
def test_handles_timeout(): assert add(1, 1) == 2   # name != behavior
```

### ✅ Corrected version (Good)
```python
# the test name reflects the behavior it verifies
```

### 🔎 Concrete detection
Test name that does not correspond to the behavior it exercises.
