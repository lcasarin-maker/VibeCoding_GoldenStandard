### Symptom
> Only runs in one environment

### Cause
Portability not tested

### Solution
Environment matrix

### Corrective Action / Prevention
Run across an environment matrix; a test that only works in one environment is not portable.


### ❌ Example of the vice (Bad)
```python
assert path == "/tmp/x"   # posix only
```

### ✅ Corrected version (Good)
```python
# environment matrix (windows / linux / mac)
```

### 🔎 Concrete detection
Tests coupled to one operating system without an environment matrix.
