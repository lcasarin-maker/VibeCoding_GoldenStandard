### Symptom
> Agent ends session without updating tasks/, HANDOFF.md, STATE.md, or writing an audit/sessions/ record

### Cause
Session close treated as optional; agent stops at task completion

### Solution
Phase 4 (Close) is required: tasks moved, HANDOFF updated, STATE updated if present, session record written, AUDIT_TRAIL appended

### Corrective Action / Prevention
DoD checklist in AGENTS.md §4 must include all session-close steps as explicit required items.

### 🔎 Concrete detection
audit.py: assert audit/sessions/ has a record for the current session; assert HANDOFF.md was updated
