### Symptom
> The skill runs commands with sudo/root or accesses private keys without authorization.

### Cause
Use of uncontrolled excessive privileges.

### Solution
Use least privilege and restrict elevated commands.

### Corrective Action / Prevention
Privileges must not silently escalate: static analysis flags indicators of unauthorized privilege escalation.


### ❌ Example of the vice (Bad)
```python
os.system("sudo rm -rf /")
```

### ✅ Corrected version (Good)
```python
os.remove(temp_file_path)
```

### 🔎 Concrete detection
Maps sudo commands and accesses to private-key files (.ssh/id_rsa, etc.).
