### Síntoma (Symptom)
> Proceso queda bloqueado sin ser detectado

### Causa (Cause)
Falta de señal de vida

### Solución (Solution)
Exigir heartbeat, timeout y política de recuperación

### Acción Correctiva / Prevención
Behavioral/doctrinal vice — not statically falsifiable in a generic way. Documented in the Golden Standard catalogs as governance knowledge; no automated test can discriminate this without human semantic judgment. Sprint 3.4 triage: reclassified from AUDITED/test_behavioral_compliance to DOC_ONLY.

### ❌ Ejemplo del vicio (Bad)
```python
resp = sock.recv(1024)        # sin timeout: si el peer muere, cuelga para siempre
```

### ✅ Versión corregida (Good)
```python
sock.settimeout(30)
resp = sock.recv(1024)        # timeout + politica de reintento/recuperacion
```

### 🔎 Detección concreta
Llamadas bloqueantes (recv/lock.acquire/join/await) sin timeout ni heartbeat
configurado.
