### Symptom
> Refactor breaks distant pieces

### Cause
Unknown causal graph

### Solution
Generate a dependency map before moving

### Corrective Action / Prevention
The wiki graph and link validator enforce that the canonical surfaces are mapped and resolved instead of being assumed.

### ❌ Example of the vice (Bad)
```bash
git mv core/auth.py util/        # without knowing who imports it -> breaks afar
```

### ✅ Corrected version (Good)
```bash
grep -rn "import core.auth" .     # map dependents BEFORE moving
git mv core/auth.py util/
```

### 🔎 Concrete detection
Moving/renaming symbols without a prior map of incoming references.
