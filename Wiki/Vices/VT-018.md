### Síntoma (Symptom)
> Mensaje cambia resultado

### Causa (Cause)
Contrato textual frágil

### Solución (Solution)
Estados estructurados

### Acción Correctiva / Prevención
Audited by DeepForensicAuditor D8 and D9 behavioral and static test validations.

### ❌ Ejemplo del vicio (Bad)
```python
assert str(err) == "Error: code 42 at line 7"   # rompe si cambia el texto
```

### ✅ Versión corregida (Good)
```python
assert err.code == 42
```

### 🔎 Detección concreta
Comparacion contra mensajes de texto exactos en vez de estados estructurados.
