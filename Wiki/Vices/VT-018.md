### Symptom
> The message changes the result

### Cause
Fragile textual contract

### Solution
Structured states

### Corrective Action / Prevention
Validate structured state, not free-text messages whose wording can silently flip the result.


### ❌ Example of the vice (Bad)
```python
assert str(err) == "Error: code 42 at line 7"   # breaks if the text changes
```

### ✅ Corrected version (Good)
```python
assert err.code == 42
```

### 🔎 Concrete detection
Comparison against exact text messages instead of structured states.
