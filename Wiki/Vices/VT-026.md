### Symptom
> Rules differ

### Cause
Different semantics

### Solution
Test an equivalent engine

### Corrective Action / Prevention
Test against an engine equivalent to production; a simplified database whose rules differ hides real failures.


### ❌ Example of the vice (Bad)
```python
# sqlite in test, postgres in prod -> constraints differ
```

### ✅ Corrected version (Good)
```python
# test against an engine equivalent to production's
```

### 🔎 Concrete detection
Test database with rules/constraints different from production.
