### Symptom
> Real text breaks it

### Cause
Incomplete alphabet

### Solution
Diverse data

### Corrective Action / Prevention
Include special characters and diverse real-world data that naive inputs never exercise.


### ❌ Example of the vice (Bad)
```python
assert slug("hello") == "hello"   # never tests accents/emojis/spaces
```

### ✅ Corrected version (Good)
```python
assert slug("Hola Munro!") == "hola-munro"
```

### 🔎 Concrete detection
Inputs with special/unicode characters are missing.
