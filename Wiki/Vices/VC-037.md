### Symptom
> No detectable regression

### Cause
Absent oracle

### Solution
Tests as a condition for change

### Corrective Action / Prevention
Behavioral/doctrinal vice — not statically falsifiable in a generic way. Documented in the Golden Standard catalogs as governance knowledge; no automated test can discriminate this without human semantic judgment. Sprint 3.4 triage: reclassified from AUDITED/test_behavioral_compliance to DOC_ONLY.

### ❌ Example of the vice (Bad)
```text
src/billing/refund.py        # 0 tests
# refund() is changed and nothing detects the regression
```

### ✅ Corrected version (Good)
```text
src/billing/refund.py
tests/billing/test_refund.py # covers the behavior before changing it
```

### 🔎 Concrete detection
Production modules without a corresponding test file (missing src/x.py ->
tests/test_x.py mapping) or without coverage of their public functions.

### 📚 External evidence
- **arxiv:2602.00409** — agents generate code without a test oracle in a measurable way; the absence of tests amplifies the risk.
