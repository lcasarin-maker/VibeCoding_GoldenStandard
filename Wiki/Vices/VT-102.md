### Symptom
> The validator accepts by name, not by behavior

### Cause
Whitelist mistaken for a test

### Solution
Every approval must revalidate active properties

### Corrective Action / Prevention
Every approval must revalidate active properties; accepting by name from a hardcoded list skips the behavior check.


### ❌ Example of the vice (Bad)
```python
APPROVED = {"a", "b"}   # fixed list, never revalidated
```

### ✅ Corrected version (Good)
```python
# every approval revalidates active properties
```

### 🔎 Concrete detection
A static approval allowlist that does not revalidate current properties.
