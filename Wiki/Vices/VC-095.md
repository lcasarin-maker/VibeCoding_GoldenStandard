### Symptom
> Irreversible risk

### Cause
Lack of a safe environment

### Solution
Separate environments

### Corrective Action / Prevention
Behavioral/doctrinal vice — not statically falsifiable in a generic way. Documented in the Golden Standard catalogs as governance knowledge; no automated test can discriminate this without human semantic judgment. Sprint 3.4 triage: reclassified from AUDITED/test_behavioral_compliance to DOC_ONLY.

### ❌ Example of the vice (Bad)
```python
# Embedded production URL/credential: any run touches prod.
DB = connect("postgres://prod-db.internal:5432", password="hunter2")
```

### ✅ Corrected version (Good)
```python
# Injected environment; prod requires an explicit, separate step.
DB = connect(os.environ["DATABASE_URL"])   # default points to sandbox
```

### 🔎 Concrete detection
Literals with production hosts/credentials (prod, internal, keys) in source
code instead of per-environment configuration.
Tools: gitleaks (secrets in code and git history), trufflehog.
