### Síntoma (Symptom)
> Riesgo irreversible

### Causa (Cause)
Falta de entorno seguro

### Solución (Solution)
Separar entornos

### Acción Correctiva / Prevención
Behavioral/doctrinal vice — not statically falsifiable in a generic way. Documented in the Golden Standard catalogs as governance knowledge; no automated test can discriminate this without human semantic judgment. Sprint 3.4 triage: reclassified from AUDITED/test_behavioral_compliance to DOC_ONLY.

### ❌ Ejemplo del vicio (Bad)
```python
# URL/credencial de produccion incrustada: cualquier corrida toca prod.
DB = connect("postgres://prod-db.internal:5432", password="hunter2")
```

### ✅ Versión corregida (Good)
```python
# Entorno inyectado; prod requiere un paso explicito y separado.
DB = connect(os.environ["DATABASE_URL"])   # default apunta a sandbox
```

### 🔎 Detección concreta
Literales con hosts/credenciales de produccion (prod, internal, claves) en codigo
fuente en vez de configuracion por entorno.
Tools: gitleaks (secretos en codigo e historial git), trufflehog.
