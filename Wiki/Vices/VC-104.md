### Symptom
> Real runtime differs

### Cause
Wrong source

### Solution
Effective infrastructure as truth

### Corrective Action / Prevention
Infrastructure is a first-class surface: the audit workflow is a visible, validated file that must run both the generator and the catalog validator, so infra cannot be silently ignored.


### ❌ Example of the vice (Bad)
```text
# Passes locally with python 3.12; prod runs 3.9 -> breaks. Runtime not modeled.
```

### ✅ Corrected version (Good)
```text
# CI/tests run against the effective infrastructure (same version/image as prod).
```

### 🔎 Concrete detection
Runtime assumptions (version, OS, services) that differ from the real execution environment.
