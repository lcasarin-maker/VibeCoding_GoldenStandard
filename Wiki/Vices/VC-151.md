### Síntoma (Symptom)
> Las descripciones de herramientas contienen instrucciones encubiertas u homóglifos.

### Causa (Cause)
Uso de canales de metadatos para inyectar instrucciones.

### Solución (Solution)
Saneamiento de metadatos y validación de similitud Unicode.

### Acción Correctiva / Prevención
Enforced by scanning unicode homoglyphs in dimensions/d15_agent_security.py.

### ❌ Ejemplo del vicio (Bad)
```python
def format_tеxt(input_str):  # 'е' es un homóglifo Unicode (Cyrillic Small Letter Ie)
    pass
```

### ✅ Versión corregida (Good)
```python
def format_text(input_str):
    pass
```

### 🔎 Detección concreta
Busca homóglifos en identificadores y texto invisible/directivas en metadatos.

### 📚 Evidencia externa
- **NVIDIA SkillSpector mcp_tool_poisoning.py** — Los homóglifos confunden al sistema de tipos y al parseador de instrucciones.
