### Symptom
> Signal not observed

### Cause
Discarded channel

### Solution
Capture error and log

### Corrective Action / Prevention
Capture and observe error output; an unobserved error stream hides the failure signal.


### ❌ Example of the vice (Bad)
```python
out = run(); assert "ok" in out.stdout   # ignores stderr and exit code
```

### ✅ Corrected version (Good)
```python
r = run(); assert r.returncode == 0 and not r.stderr
```

### 🔎 Concrete detection
Tests that ignore stderr or the process exit code.
