### Symptom
> The auditor approves itself

### Cause
Non-independent verifier

### Solution
External oracle

### Corrective Action / Prevention
A test must be judged against an external oracle, never approve itself in a circular loop.


### ❌ Example of the vice (Bad)
```python
assert auditor.audit(auditor) == "ok"   # evaluates itself
```

### ✅ Corrected version (Good)
```python
assert auditor.audit(subject) == expected_from_spec
```

### 🔎 Concrete detection
The subject under test is also its own oracle (self-approval).

- **[[Evidence/arxiv2602_00409|arxiv:2602.00409]]** — an empirical study confirms that agents generate tests without an oracle (no assert, tautological, over-mocked) in a measurable way.
