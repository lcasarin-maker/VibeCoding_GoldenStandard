### Síntoma (Symptom)
> Auditor se aprueba

### Causa (Cause)
Verificador no independiente

### Solución (Solution)
Oráculo externo

### Acción Correctiva / Prevención
Audited by DeepForensicAuditor D8 and D9 behavioral and static test validations.

### ❌ Ejemplo del vicio (Bad)
```python
assert auditor.audit(auditor) == "ok"   # se evalua a si mismo
```

### ✅ Versión corregida (Good)
```python
assert auditor.audit(subject) == expected_from_spec
```

### 🔎 Detección concreta
El sujeto bajo prueba es tambien su propio oraculo (auto-aprobacion).
