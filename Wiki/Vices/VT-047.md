### Síntoma (Symptom)

### Causa (Cause)

### Solución (Solution)
Pruebas de volumen

### Acción Correctiva / Prevención
Volumen real: test ejercita 1000+ sesiones (lo opuesto al dataset pequeño).

### ❌ Ejemplo del vicio (Bad)
```python
# 'Evaluacion' sobre un unico ejemplo: ruido estadistico, no senal.
def test_model():
    assert model.predict([sample]) == [1]
```

### ✅ Versión corregida (Good)
```python
# Conjunto representativo + umbral de metrica acordado.
def test_model_accuracy():
    acc = evaluate(model, validation_set)   # N>=200 ejemplos
    assert acc >= 0.85
```

### 🔎 Detección concreta
Tests de modelo/heuristica que afirman sobre 1-2 muestras fijas en lugar de una
metrica agregada sobre un dataset.
