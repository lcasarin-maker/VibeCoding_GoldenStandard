### Symptom
> Scale breaks it

### Cause
Complexity not exercised

### Solution
Volume tests

### Corrective Action / Prevention
Real volume: the test exercises 1000+ sessions (the opposite of a small dataset).

### ❌ Example of the vice (Bad)
```python
# 'Evaluation' over a single example: statistical noise, not signal.
def test_model():
    assert model.predict([sample]) == [1]
```

### ✅ Corrected version (Good)
```python
# Representative set + agreed metric threshold.
def test_model_accuracy():
    acc = evaluate(model, validation_set)   # N>=200 examples
    assert acc >= 0.85
```

### 🔎 Concrete detection
Model/heuristic tests that assert on 1-2 fixed samples instead of an
aggregate metric over a dataset.
