### Síntoma (Symptom)
> El valor esperado termina copiado dentro del código o del fixture y deja de ser una observación independiente

### Causa (Cause)
El oráculo se incrusta en la implementación o en una fixture compartida

### Solución (Solution)
Mantener el esperado fuera de la ruta productiva y validar contra una especificación externa

### Acción Correctiva / Prevención
Audited as oracle-contamination in test fixtures or implementation glue; separate from stale-spec mismatches.

### ❌ Ejemplo del vicio (Bad)
```python
EXPECTED = compute(x)
assert compute(x) == EXPECTED   # el esperado sale del propio SUT
```

### ✅ Versión corregida (Good)
```python
assert compute(x) == spec_value(x)   # esperado de una especificacion externa
```

### 🔎 Detección concreta
El valor esperado se deriva del propio SUT en vez de una fuente independiente.
