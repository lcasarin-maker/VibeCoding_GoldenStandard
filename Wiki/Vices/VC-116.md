# VC-116: Instalación automática de dependencias no verificadas

| Campo | Detalle |
|---|---|
| **ID** | `VC-116` |
| **Categoría** | Vibe Coding |
| **Estado** | **REMEDIATED** |
| **Severidad** | **high** |
| **Tags** | `vibe-coding`, `remediated` |
| **Mecanismo de Validación** | `test_import_error_guard_does_not_pip_install` |

---

### Síntoma (Symptom)
> Un `ImportError` desencadena `pip install` sin revisión humana

### Causa (Cause)
El error handler delega decisiones de supply chain al entorno sin control

### Solución (Solution)
Prohibir `pip install` automático; reportar la dependencia faltante con guía de instalación manual y detener; el humano decide qué instala

### Acción Correctiva / Prevención
Disabled automatic subprocess pip installs in repair_failing_tests.py, forcing manual package guidance.
