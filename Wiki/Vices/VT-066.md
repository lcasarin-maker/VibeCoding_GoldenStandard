### Symptom
> Files never run

### Cause
Inventory not compared

### Solution
Reconcile the inventory

### Corrective Action / Prevention
Reconcile the test inventory so orphan files that are never discovered or run are surfaced.


### ❌ Example of the vice (Bad)
```python
# Test that exercises already-removed code: passes via mock, tests nothing real.
def test_old_engine():
    eng = Mock(spec=OldEngine)   # OldEngine no longer exists in src/
    assert eng.run() is not None
```

### ✅ Corrected version (Good)
```python
# With no corresponding production code, the test is deleted (or rewritten over the new one).
def test_engine_runs():
    assert Engine().run().status == "ok"
```

### 🔎 Concrete detection
Detect tests whose symbol under test (import/Mock spec) no longer exists in the
production tree = orphan test.
