### Síntoma (Symptom)
> Archivos nunca corren

### Causa (Cause)
Inventario no comparado

### Solución (Solution)
Reconciliar inventario

### Acción Correctiva / Prevención
Discovery gaps prevented by run_compliance_tests executing full pytest tests/ dynamically, verified in test_infrastructure.py.

### ❌ Ejemplo del vicio (Bad)
```python
# Test que ejercita codigo ya eliminado: pasa por mock, no prueba nada real.
def test_old_engine():
    eng = Mock(spec=OldEngine)   # OldEngine ya no existe en src/
    assert eng.run() is not None
```

### ✅ Versión corregida (Good)
```python
# Sin codigo de produccion correspondiente, el test se borra (o se reescribe sobre el nuevo).
def test_engine_runs():
    assert Engine().run().status == "ok"
```

### 🔎 Detección concreta
Detectar tests cuyo simbolo bajo prueba (import/Mock spec) ya no existe en el arbol
de produccion = test huerfano.
