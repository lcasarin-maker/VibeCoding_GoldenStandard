### Síntoma (Symptom)
> Funcionamiento depende de una ubicación fija

### Causa (Cause)
Estado externo codificado

### Solución (Solution)
Resolver ubicaciones por contrato configurable

### Acción Correctiva / Prevención
Rutas absolutas hardcodeadas (C:\, /home/, /Users/) detectadas por escaneo regex discriminante.

### ❌ Ejemplo del vicio (Bad)
```python
# Ruta absoluta de la maquina del agente: rompe en cualquier otro entorno.
LOG = open("C:/Users/Luis/AI/project/out.log", "w")
```

### ✅ Versión corregida (Good)
```python
# Ruta derivada de configuracion/entorno y relativa a la raiz del proyecto.
from pathlib import Path
import os
LOG = open(Path(os.environ.get("PROJECT_ROOT", ".")) / "out.log", "w")
```

### 🔎 Detección concreta
Regex/AST: literales de ruta que contengan 'C:\\', '/Users/', '/home/' o
separadores de unidad de Windows en codigo de aplicacion.
