### Symptom
> The error disappears

### Cause
Collapsed failure

### Solution
Fail explicitly

### Corrective Action / Prevention
Enforced by D5 (Angry Path) AST TryBlockVisitor flagging empty try-except blocks or silent pass/continue statements.

### ❌ Example of the vice (Bad)
```python
def test_parse():
    try:
        parse(bad_input)
    except Exception:
        pass   # absorbs the failure: the test never fails
```

### ✅ Corrected version (Good)
```python
def test_parse():
    with pytest.raises(ValueError):
        parse(bad_input)   # asserts the expected exception
```

### 🔎 Concrete detection
AST: try/except blocks with a 'pass' body inside test_* functions, which hide
any exception.
Tools: ruff BLE001 (blind-except), flake8-bugbear, pylint W0702.
