### Síntoma (Symptom)
> Error desaparece

### Causa (Cause)
Fallo colapsado

### Solución (Solution)
Fallar explícitamente

### Acción Correctiva / Prevención
Enforced by D5 (Angry Path) AST TryBlockVisitor flagging empty try-except blocks or silent pass/continue statements.

### ❌ Ejemplo del vicio (Bad)
```python
def test_parse():
    try:
        parse(bad_input)
    except Exception:
        pass   # absorbe el fallo: el test nunca falla
```

### ✅ Versión corregida (Good)
```python
def test_parse():
    with pytest.raises(ValueError):
        parse(bad_input)   # afirma la excepcion esperada
```

### 🔎 Detección concreta
AST: bloques try/except con cuerpo 'pass' dentro de funciones test_*, que ocultan
cualquier excepcion.

### 📚 Evidencia externa
- **arxiv:2602.00409** — estudio empirico confirma que los agentes generan tests sin oraculo (sin assert, tautologicos o sobre-mockeados) de forma medible.
