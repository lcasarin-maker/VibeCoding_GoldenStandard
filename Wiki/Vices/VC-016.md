### Symptom
> Only what was asked for is known

### Cause
Absent mental model

### Solution
Mandatory explainability

### Corrective Action / Prevention
Behavioral/doctrinal vice — not statically falsifiable in a generic way. Documented in the Golden Standard catalogs as governance knowledge; no automated test can discriminate this without human semantic judgment. Sprint 3.4 triage: reclassified from AUDITED/test_behavioral_compliance to DOC_ONLY.

### ❌ Example of the vice (Bad)
```python
# High cyclomatic-complexity function, with no docstring or mental model.
def proc(d):
    r = []
    for k in d:
        if k > 0:
            for j in d[k]:
                if j and (j % 2 or k < 3) and not seen(j): r.append(f(j, k))
    return r
```

### ✅ Corrected version (Good)
```python
# Decomposed into named, explainable units; low complexity.
def proc(d):
    return [transform(j, k) for k, j in relevant_pairs(d)]
```

### 🔎 Concrete detection
Cyclomatic complexity > threshold (e.g. 10) or long functions without a docstring
(radon / mccabe / ruff C901).
