### Symptom
> The calendar breaks it

### Cause
Temporal boundaries

### Solution
Calendar cases

### Corrective Action / Prevention
Cover calendar boundaries such as leap days and month/year edges; ideal dates hide date-arithmetic bugs.


### ❌ Example of the vice (Bad)
```python
@pytest.mark.skip("fix later")   # no deadline: eternal
def test_export(): ...
```

### ✅ Corrected version (Good)
```python
# Every skip carries an expiration; a CI check fails if the date has passed.
@pytest.mark.skip("BUG-55, expires 2026-07-01")
def test_export(): ...
```

### 🔎 Concrete detection
Test linter: skip/xfail without a parseable expiration date, or with an already-passed date.
