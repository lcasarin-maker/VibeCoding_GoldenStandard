### Symptom
> It says all, runs few

### Cause
Untraceable report

### Solution
Report real domains

### Corrective Action / Prevention
Report the domains actually exercised; a pass report claiming all while running few is a lie.


### ❌ Example of the vice (Bad)
```text
# The report says 'passed' for domains that were skipped.
```

### ✅ Corrected version (Good)
```text
# report only the domains actually executed
```

### 🔎 Concrete detection
Reports that mark omitted or skipped domains as 'passed'.
