### Symptom
> Success is declared without logs

### Cause
Confusion between assertion and verification

### Solution
Forbid verdicts without observable evidence

### Corrective Action / Prevention
Empirical proof is required before declaring success: closure fails when no structured evidence artifact backs the claim, forcing verification over optimism.


### ❌ Example of the vice (Bad)
```python
# Success is declared without any observable evidence.
def deploy():
    do_release()
    print("Deploy OK")  # assertion, not verification
```

### ✅ Corrected version (Good)
```python
# The verdict requires evidence: the flow fails if there is no proof.
def deploy():
    result = do_release()
    assert result.health_check().ok, "deploy without green health-check"
    write_evidence(".protocol/evidence/deploy.json", result.metrics())
```

### 🔎 Concrete detection
Flag as a vice any path that emits 'OK'/'success'/'done' without an assert over
observable state or an evidence artifact generated in the same execution.
