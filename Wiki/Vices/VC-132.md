### Symptom
> A vague prompt produces functionally correct but semantically wrong code, which feeds later prompts and amplifies the error downstream.


### Cause
Specification ambiguity unresolved before generating; the wrong interpretation propagates to each subsequent step.


### Solution
Resolve the ambiguity before generating (goal, constraints, examples) and validate the interpretation at each branch point.


### Corrective Action / Prevention
Documented; enforcement downstream. Recipe: confirm the interpretation at branch points before building on it.


### ❌ Example of the vice (Bad)
```text
# Prompt: 'make it fast'. The agent caches the wrong layer.
# The next 5 prompts build on that misplaced cache.
```

### ✅ Corrected version (Good)
```text
# Resolve the ambiguity first: 'fast' = p95 < 200ms on endpoint X.
# The interpretation is validated before it feeds later steps.
```

### 🔎 Concrete detection
A chain of changes derived from an ambiguous spec without an interpretation-validation point; accumulated semantic divergence. Extends VC-025 (vague spec) to the cascade effect.
