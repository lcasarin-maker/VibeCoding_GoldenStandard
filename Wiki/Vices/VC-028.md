### Symptom
> External changes break the core

### Cause
Temporal coupling

### Solution
A core with stable dependencies

### Corrective Action / Prevention
Static detector `vc028_core_dependent_on_unstable_parts` flags core imports from experimental or unstable modules.

### ❌ Example of the vice (Bad)
```python
# The core imports from a module marked experimental/volatile.
from experimental.newapi import parse   # core depends on something unstable
```

### ✅ Corrected version (Good)
```python
# The core depends only on stable interfaces.
from core.stable.parser import parse
```

### 🔎 Concrete detection
Core modules (core/) that import from packages marked experimental/unstable/
beta or with high churn.
