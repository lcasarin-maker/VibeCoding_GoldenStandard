### Symptom
> External changes break the core

### Cause
Temporal coupling

### Solution
A core with stable dependencies

### Corrective Action / Prevention
Behavioral/doctrinal vice — not statically falsifiable in a generic way. Documented in the Golden Standard catalogs as governance knowledge; no automated test can discriminate this without human semantic judgment. Sprint 3.4 triage: reclassified from AUDITED/test_behavioral_compliance to DOC_ONLY.

### ❌ Example of the vice (Bad)
```python
# The core imports from a module marked experimental/volatile.
from experimental.newapi import parse   # core depends on something unstable
```

### ✅ Corrected version (Good)
```python
# The core depends only on stable interfaces.
from core.stable.parser import parse
```

### 🔎 Concrete detection
Core modules (core/) that import from packages marked experimental/unstable/
beta or with high churn.
