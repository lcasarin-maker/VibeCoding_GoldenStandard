### Symptom
> Slowness passes

### Cause
Poorly calibrated threshold

### Solution
Functional limits

### Corrective Action / Prevention
Assert functional limits, not just a generous timeout under which slowness passes.


### ❌ Example of the vice (Bad)
```python
@timeout(30)   # arbitrary 30s hides real slowness
```

### ✅ Corrected version (Good)
```python
# justified functional limits (p95 < X)
```

### 🔎 Concrete detection
Arbitrary timeouts that mask slowness instead of functional limits.
