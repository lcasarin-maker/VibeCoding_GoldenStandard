### Symptom
> OK without checking

### Cause
Non-causal narrative

### Solution
Log after verification

### Corrective Action / Prevention
Log success only after the verification that justifies it, never before checking.


### ❌ Example of the vice (Bad)
```python
log("done"); save()   # success log before verifying
```

### ✅ Corrected version (Good)
```python
save(); assert persisted(); log("done")   # log after verifying
```

### 🔎 Concrete detection
Logging success before verifying the result.
