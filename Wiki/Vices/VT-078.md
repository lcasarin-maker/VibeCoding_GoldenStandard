### Symptom
> Depends on installation

### Cause
Non-portable environment

### Solution
Reproducibility

### Corrective Action / Prevention
Tests must be reproducible off one developer's machine, independent of a local installation.


### ❌ Example of the vice (Bad)
```text

```

### ✅ Corrected version (Good)
```text
# reproducible in a container / clean CI
```

### 🔎 Concrete detection
Tests reproducible only on the author's local machine.
