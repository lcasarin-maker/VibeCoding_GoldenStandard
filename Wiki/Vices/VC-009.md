### Symptom
> Prior integrity is lost

### Cause
Non-reversible operation

### Solution
Prefer minimal diffs

### Corrective Action / Prevention
Full rewrite vs surgical edit: behavioral judgment vice (when a rewrite is justified). No static test discriminates intent, so this stays a manual-review DOC_ONLY entry.

### ❌ Example of the vice (Bad)
```text
# The agent rewrites the whole file to change two lines.
# diff: 198 deletions(-), 200 insertions(+)  -> history and invariants are lost
```

### ✅ Corrected version (Good)
```text
# Minimal, surgical diff: only what changes.
- timeout = 30
+ timeout = 60
```

### 🔎 Concrete detection
Diff heuristic: changes where >80% of an existing file's lines are
deleted and reinserted = total rewrite disguised as an edit.
