### Symptom
> Only the ideal case

### Cause
Poor domain

### Solution
Adverse cases

### Corrective Action / Prevention
Cover adverse cases, not just the ideal happy path.


### ❌ Example of the vice (Bad)
```python
def test(): assert div(10, 2) == 5   # only the happy case
```

### ✅ Corrected version (Good)
```python
# include div(1, 0), negatives, overflow
```

### 🔎 Concrete detection
Coverage only of the happy path; adverse cases are missing.
