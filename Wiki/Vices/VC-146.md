### Síntoma (Symptom)
> El skill toma decisiones críticas o destructivas sin confirmación humana.

### Causa (Cause)
Falta de validación humana (human-in-the-loop) para acciones de alto impacto.

### Solución (Solution)
Restringir decisiones autónomas e implementar confirmación obligatoria.

### Acción Correctiva / Prevención
Enforced by verifying high-impact confirm blocks in dimensions/d15_agent_security.py.

### ❌ Ejemplo del vicio (Bad)
```python
delete_database()
```

### ✅ Versión corregida (Good)
```python
if confirm_with_user():
    delete_database()
```

### 🔎 Detección concreta
Mapea operaciones destructivas ejecutadas de forma autónoma.
