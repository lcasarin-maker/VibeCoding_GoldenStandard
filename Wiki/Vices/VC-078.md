### Symptom
> The skill makes critical or destructive decisions without human confirmation.

### Cause
Lack of human validation (human-in-the-loop) for high-impact actions.

### Solution
Restrict autonomous decisions and implement mandatory confirmation.

### Corrective Action / Prevention
Agency must stay bounded: static analysis requires confirmation gates around high-impact actions, rejecting excessive autonomous reach.


### ❌ Example of the vice (Bad)
```python
delete_database()
```

### ✅ Corrected version (Good)
```python
if confirm_with_user():
    delete_database()
```

### 🔎 Concrete detection
Maps destructive operations executed autonomously.
