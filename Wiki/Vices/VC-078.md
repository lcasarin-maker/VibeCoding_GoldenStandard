### Symptom
> A marker reaches production

### Cause
Incomplete state accepted

### Solution
Block active placeholders

### Corrective Action / Prevention
Detected by D2 (completeness) via AST: empty function bodies, stub docstrings, and pass-only functions are rejected. D9 (test purity) rejects tests that have no active assertions. Upgrade 2026-06-04: VC-078 leaves DOC_ONLY because D2 and D9 cover the pattern mechanically in scripts/ and tests/.

### ❌ Example of the vice (Bad)
```python
API_KEY = "REPLACE_ME"           # placeholder that reaches production
```

### ✅ Corrected version (Good)
```python
API_KEY = os.environ["API_KEY"]  # fails if missing; no placeholder
```

### 🔎 Concrete detection
Placeholder literals (REPLACE_ME / XXX / changeme / foo) in production paths.
