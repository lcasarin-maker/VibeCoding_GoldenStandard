### Symptom
> Text replaces a structured API

### Cause
Fragile representation

### Solution
Use structured operations

### Corrective Action / Prevention
Static detector `vc070_blind_shell_edit` flags shell-based mutation of structured files; use a parser instead of sed/awk/grep-based edits.

### ❌ Example of the vice (Bad)
```bash
# Editing JSON with a stream editor: fragile, breaks on the first odd case.
sed -i 's/"port": 80/"port": 443/' config.json
```

### ✅ Corrected version (Good)
```bash
# Structured operation with a real parser for the format.
python -c "import json; d=json.load(open('config.json')); d['port']=443; json.dump(d, open('config.json','w'), indent=2)"
```

### 🔎 Concrete detection
Use of sed/awk/grep to mutate structured files (.json/.yaml/.toml/.xml) instead
of a parser for the format.
