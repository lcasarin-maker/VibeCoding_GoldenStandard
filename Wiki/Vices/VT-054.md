### Symptom
> Nobody runs them

### Cause
Voluntariness

### Solution
Mandatory by risk

### Corrective Action / Prevention
Make risk-bearing tests mandatory; optional tests are not run and give no protection.


### ❌ Example of the vice (Bad)
```python
@pytest.mark.optional   # skipped when in a hurry
```

### ✅ Corrected version (Good)
```python
# mandatory by risk (without an optional marker)
```

### 🔎 Concrete detection
Tests marked optional that cover risk and are skipped under pressure.

### 📚 External evidence
- **arxiv:2602.00409** — an empirical study confirms that agents generate tests without an oracle (no assert, tautological, over-mocked) in a measurable way.
