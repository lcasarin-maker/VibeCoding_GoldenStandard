### Symptom
> Does not run

### Cause
Fictitious coverage

### Solution
Invalid global result if it omits a critical test

### Corrective Action / Prevention
A permanently skipped critical test invalidates the global result; skipping must not silently omit it.


### ❌ Example of the vice (Bad)
```python
@pytest.mark.skip   # permanently skipped, without condition or reason
def test_payment():
    ...
```

### ✅ Corrected version (Good)
```python
@pytest.mark.skipif(no_sandbox(), reason="requires a payments sandbox, BUG-77")
def test_payment():
    ...
```

### 🔎 Concrete detection
Detect unconditional @skip without a reason or expiration date; prefer skipif with
a verifiable condition.
