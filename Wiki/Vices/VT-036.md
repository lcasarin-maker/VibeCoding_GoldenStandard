### Síntoma (Symptom)
> No corre

### Causa (Cause)
Cobertura ficticia

### Solución (Solution)
Resultado global inválido si omite prueba crítica

### Acción Correctiva / Prevención
Checked by D9 Test Purity which rejects permanent xfail or skip markers unless annotated with removal criteria or reasons.

### ❌ Ejemplo del vicio (Bad)
```python
@pytest.mark.skip   # saltado permanentemente, sin condicion ni motivo
def test_payment():
    ...
```

### ✅ Versión corregida (Good)
```python
@pytest.mark.skipif(no_sandbox(), reason="requiere sandbox de pagos, BUG-77")
def test_payment():
    ...
```

### 🔎 Detección concreta
Detectar @skip incondicional sin reason ni fecha de expiracion; preferir skipif con
condicion verificable.
