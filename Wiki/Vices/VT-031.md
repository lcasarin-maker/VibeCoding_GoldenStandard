### Symptom

### Cause
Simulated action

### Solution
Verify the external effect

### Corrective Action / Prevention
Verify the external effect of a command, not a stub that merely prints success.


### ❌ Example of the vice (Bad)
```python
run = Mock()   # the command's external effect is not verified
```

### ✅ Corrected version (Good)
```python
# run in a sandbox and verify the effect (file created, exit code)
```

### 🔎 Concrete detection
Command/subprocess stub without verifying its external effect.
