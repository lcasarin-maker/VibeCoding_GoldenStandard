### Síntoma (Symptom)
> Texto sustituye API estructurada

### Causa (Cause)
Representación frágil

### Solución (Solution)
Usar operaciones estructuradas

### Acción Correctiva / Prevención
Behavioral/doctrinal vice — not statically falsifiable in a generic way. Documented in the Golden Standard catalogs as governance knowledge; no automated test can discriminate this without human semantic judgment. Sprint 3.4 triage: reclassified from AUDITED/test_behavioral_compliance to DOC_ONLY.

### ❌ Ejemplo del vicio (Bad)
```bash
# Editar JSON con sed: fragil, rompe el formato al primer caso raro.
sed -i 's/"port": 80/"port": 443/' config.json
```

### ✅ Versión corregida (Good)
```bash
# Operacion estructurada con un parser real del formato.
python -c "import json; d=json.load(open('config.json')); d['port']=443; json.dump(d, open('config.json','w'), indent=2)"
```

### 🔎 Detección concreta
Uso de sed/awk/grep para mutar archivos estructurados (.json/.yaml/.toml/.xml) en
vez de un parser del formato.
