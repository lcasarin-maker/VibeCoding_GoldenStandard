### Síntoma (Symptom)
> Fallos de validación al migrar a entornos secundarios

### Causa (Cause)
Uso de rutas de almacenamiento absoluto de la máquina de origen

### Solución (Solution)
Parametrizar dinámicamente recursos mediante inyección o variables lógicas

### Acción Correctiva / Prevención
Enforced by D9 absolute path scanners. In code, resolved via dynamic str(Path(__file__).resolve().parent.parent) path bootstrapping.

### ❌ Ejemplo del vicio (Bad)
```python
# Afirma sobre el id/orden fisico de memoria, no sobre el contenido.
def test_cache():
    assert id(cache.get("k")) == 140234567   # fragil y sin sentido
```

### ✅ Versión corregida (Good)
```python
def test_cache():
    assert cache.get("k") == expected_value
```

### 🔎 Detección concreta
Asserts sobre id(), direcciones de memoria, hash() no determinista u orden de
iteracion no garantizado.

### 📚 Evidencia externa
- **arxiv:2602.00409** — estudio empirico confirma que los agentes generan tests sin oraculo (sin assert, tautologicos o sobre-mockeados) de forma medible.
