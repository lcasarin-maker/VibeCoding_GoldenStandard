### Symptom
> Validation failures when migrating to secondary environments

### Cause
Use of absolute storage paths from the origin machine

### Solution
Dynamically parametrize resources via injection or logical variables

### Corrective Action / Prevention
Enforced by D9 absolute path scanners. In code, resolved via dynamic str(Path(__file__).resolve().parent.parent) path bootstrapping.

### ❌ Example of the vice (Bad)
```python
# Asserts on the physical memory id/order, not on the content.
def test_cache():
    assert id(cache.get("k")) == 140234567   # fragile and meaningless
```

### ✅ Corrected version (Good)
```python
def test_cache():
    assert cache.get("k") == expected_value
```

### 🔎 Concrete detection
Asserts on id(), memory addresses, non-deterministic hash(), or non-guaranteed
iteration order.
