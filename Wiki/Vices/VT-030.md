### Symptom
> The real system is not tested

### Cause
Invasive substitution

### Solution
Limit patches

### Corrective Action / Prevention
Limit patching to the boundary; over-patching replaces the system under test with the test's own assumptions.


### ❌ Example of the vice (Bad)
```python
monkeypatch.setattr(mod, "validate", lambda x: True)   # patches what is being tested
```

### ✅ Corrected version (Good)
```python
# limit patches to external dependencies, never to the subject
```

### 🔎 Concrete detection
Monkeypatch that replaces the very function under test.

- **arxiv:2602.00409** — an empirical study confirms that agents generate tests without an oracle (no assert, tautological, over-mocked) in a measurable way.
