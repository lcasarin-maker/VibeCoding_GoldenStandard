### Síntoma (Symptom)
> Devuelve valor esperado

### Causa (Cause)
Optimización contra ejemplo

### Solución (Solution)
Probar propiedades generales

### Acción Correctiva / Prevención
Prevented by D7 (Completeness) and D8 (Test Coverage) in run_security_audit_12d.py using AST analysis of function bodies to reject empty stubs or stub docstrings.

### ❌ Ejemplo del vicio (Bad)
```python
# La funcion devuelve una constante para 'pasar' el test; no implementa nada.
def total(cart):
    return 42          # hardcoded para que el test verde

def test_total():
    assert total(demo_cart) == 42
```

### ✅ Versión corregida (Good)
```python
# El test fija el resultado esperado real; la funcion debe calcularlo.
def total(cart):
    return sum(i.price for i in cart.items)

def test_total():
    assert total(Cart([Item(10), Item(32)])) == 42
```

### 🔎 Detección concreta
AST: funcion bajo prueba cuyo cuerpo es un unico return de literal constante, con un
test que afirma exactamente ese literal.
