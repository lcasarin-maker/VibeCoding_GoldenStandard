### Symptom
> The test breaks the runner

### Cause
Global mutation

### Solution
Isolate test effects

### Corrective Action / Prevention
Isolate each test's side effects so one cannot break the runner or other tests.


### ❌ Example of the vice (Bad)
```python
# a global patch is not reverted and contaminates other tests
```

### ✅ Corrected version (Good)
```python
# isolate effects per test (teardown / scoped fixture)
```

### 🔎 Concrete detection
Global capture/patch that is not reverted and leaks effects between tests.
