### Symptom
> It tests an external library, not the system

### Cause
Wrong subject

### Solution
Fail real components

### Corrective Action / Prevention
Exercise the real components of the system, not an external library standing in for them.


### ❌ Example of the vice (Bad)
```python
# a mock is 'failed', not a real component
```

### ✅ Corrected version (Good)
```python
# fail real components (kill a pod, cut the network)
```

### 🔎 Concrete detection
Chaos tests that affect mocked components, not real ones.
