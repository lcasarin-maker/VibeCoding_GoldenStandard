### Symptom
> Only passes the test's input

### Cause
Memorization

### Solution
Vary the inputs

### Corrective Action / Prevention
Audited by DeepForensicAuditor D8 and D9 behavioral and static test validations.

### ❌ Example of the vice (Bad)
```python
def classify(s): return "spam" if s == "BUY NOW" else "ham"  # only the test's input
```

### ✅ Corrected version (Good)
```python
def classify(s): return model.predict(s)   # generalizes; the test uses several inputs
```

### 🔎 Concrete detection
Implementation that only succeeds for the test's exact input (memorizes the case).
