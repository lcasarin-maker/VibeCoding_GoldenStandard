### Symptom
> Only passes the test's input

### Cause
Memorization

### Solution
Vary the inputs

### Corrective Action / Prevention
A test must generalize beyond the one input it was written for; an assertion that only holds for a single hardcoded datum proves nothing.


### ❌ Example of the vice (Bad)
```python
def classify(s): return "spam" if s == "BUY NOW" else "ham"  # only the test's input
```

### ✅ Corrected version (Good)
```python
def classify(s): return model.predict(s)   # generalizes; the test uses several inputs
```

### 🔎 Concrete detection
Implementation that only succeeds for the test's exact input (memorizes the case).
