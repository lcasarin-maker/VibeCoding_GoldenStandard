### Symptom
> The text exists

### Cause
Presence by action

### Solution
Invoke the mandate

### Corrective Action / Prevention
Invoke the real behavior the mandate requires; asserting that some text exists is ceremony.


### ❌ Example of the vice (Bad)
```python
# Weak textual assertion: passes with almost any output.
def test_report():
    r = build_report()
    assert "ok" in str(r).lower()   # theatrical
```

### ✅ Corrected version (Good)
```python
def test_report():
    r = build_report()
    assert r.total == 100 and r.errors == []
```

### 🔎 Concrete detection
Flag asserts that only check vague substrings ('ok','success','done') over
str(result) instead of structured fields.
