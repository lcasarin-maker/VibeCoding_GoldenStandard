### Síntoma (Symptom)
> Texto existe

### Causa (Cause)
Presencia por acción

### Solución (Solution)
Invocar mandato

### Acción Correctiva / Prevención
Checked by D9 (Test Purity) using AST TestTheaterVisitor to flag assert True, assertTrue(True), assertEqual(x, x), and tests without active asserts.

### ❌ Ejemplo del vicio (Bad)
```python
# Asercion textual debil: pasa con casi cualquier salida.
def test_report():
    r = build_report()
    assert "ok" in str(r).lower()   # teatral
```

### ✅ Versión corregida (Good)
```python
def test_report():
    r = build_report()
    assert r.total == 100 and r.errors == []
```

### 🔎 Detección concreta
Marcar asserts que solo comprueban subcadenas vagas ('ok','success','done') sobre
str(result) en vez de campos estructurados.
