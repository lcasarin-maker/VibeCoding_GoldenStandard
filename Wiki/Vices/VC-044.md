### Symptom
> Operation fails due to external limits

### Cause
Unbudgeted finite resource

### Solution
Model limits, backoff, and degradation before executing

### Corrective Action / Prevention
Quotas must be visible up front: topology validation requires the tokenomics index to expose its subindices and map before the category is treated as complete.


### ❌ Example of the vice (Bad)
```python
for i in range(10000): call_api(i)   # blows up on rate limit, no handling
```

### ✅ Corrected version (Good)
```python
for i in range(10000):
    call_with_backoff(call_api, i)   # 429 -> modeled retry/backoff
```

### 🔎 Concrete detection
Calls to APIs with limits without backoff/retry/degradation modeled beforehand.
