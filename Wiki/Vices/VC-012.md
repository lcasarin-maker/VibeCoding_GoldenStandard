### Cause

### Solution

### Corrective Action / Prevention

### ❌ Example of the vice (Bad)
```python
def pay(x):
    return charge(x)   # shortcut: no idempotency or retries (unrecorded)
```

### ✅ Corrected version (Good)
```python
# The debt is declared explicitly in a trackable register.
def pay(x):
    return charge(x)   # DEBT-21: missing idempotency, see TECH_DEBT.md
```

entry in a technical-debt register.
