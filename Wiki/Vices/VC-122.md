### Síntoma (Symptom)
> Instalación automática de paquetes arbitrarios o importaciones dinámicas que alteran el entorno sin consentimiento.

### Causa (Cause)
Falta de control preventivo en los hooks de pre-ejecución.

### Solución (Solution)
Bloquear de forma física cualquier comando que intente realizar descargas de paquetes o ejecuciones de urls no declaradas de forma explícita y aprobadas.

### Acción Correctiva / Prevención
pip install automático por subprocess en scripts/ bloqueado por detector + repo-guard de ruta activa.

### ❌ Ejemplo del vicio (Bad)
```bash
# Ejecuta un script remoto sin fijar ni verificar: el proveedor puede inyectar codigo.
curl -s https://example.com/install.sh | bash
```

### ✅ Versión corregida (Good)
```bash
# Descarga, fija por hash y verifica antes de ejecutar.
curl -sSfL https://example.com/install.sh -o install.sh
echo "<sha256-conocido>  install.sh" | sha256sum -c -
bash install.sh
```

### 🔎 Detección concreta
Detectar patrones 'curl|bash', postinstall scripts no fijados, o ejecucion de
binarios descargados sin verificacion de hash/firma.
