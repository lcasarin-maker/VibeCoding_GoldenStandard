### Symptom
> Infinite debt

### Cause
No expiration

### Solution
Mandatory deadline

### Corrective Action / Prevention
A deferred skip needs a hard deadline; an open-ended skip becomes permanent debt.


### ❌ Example of the vice (Bad)
```python
@pytest.mark.skip("skip later, we'll look at it afterward")   # 'afterward' never comes
def test_webhook(): ...
```

### ✅ Corrected version (Good)
```python
# If it cannot run now, it becomes backlog with a ticket, not a mute skip.
@pytest.mark.skipif(not WEBHOOK_URL, reason="set WEBHOOK_URL; track in BUG-91")
def test_webhook(): ...
```

### 🔎 Concrete detection
Detect skip reasons with deferring language ('later','afterward','TODO') without
an associated ticket.
