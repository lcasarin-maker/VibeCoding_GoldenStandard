### Acción Correctiva / Prevención
Checked by D9 Test Purity which rejects permanent xfail or skip markers unless annotated with removal criteria or reasons.

### ❌ Ejemplo del vicio (Bad)
```python
@pytest.mark.skip("skip luego, lo vemos despues")   # 'despues' nunca llega
def test_webhook(): ...
```

### ✅ Versión corregida (Good)
```python
# Si no se puede correr ahora, se convierte en backlog con ticket, no en skip mudo.
@pytest.mark.skipif(not WEBHOOK_URL, reason="set WEBHOOK_URL; track en BUG-91")
def test_webhook(): ...
```

### 🔎 Detección concreta
Detectar reasons de skip con lenguaje aplazante ('luego','despues','TODO') sin
ticket asociado.
